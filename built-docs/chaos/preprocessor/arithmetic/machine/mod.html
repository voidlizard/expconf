CHAOS_PP_MOD_MACHINE

The CHAOS_PP_MOD_MACHINE macro computes the remainder produced by dividing two values.

Usage - C99 Specific 

CHAOS_PP_MOD_MACHINE(x, y)
x

The base. Valid values range from 0 to CHAOS_PP_LIMIT_MAG.

y

The modulus. Valid values range from 1 to CHAOS_PP_LIMIT_MAG.

Remarks

This macro expands to the remainder produced by dividing x by y. If y is 0, this macro intentionally produces an error.

This macro behaves identically to CHAOS_PP_MOD except that it is implemented with the continuation machine, which can yield significant performance improvements.

This macro may not be used inside a continuation machine instruction. However, the instruction upon which this macro is built, (0xchaos)(0xmod), may be used inside a continuation. 

CHAOS_PP_MACHINE(, 22, (0xchaos)(0xmod), 7, (0xchaos)(0xstop),) // 1

CHAOS_PP_MOD_MACHINE_ID

The CHAOS_PP_MOD_MACHINE_ID macro indirectly expands to CHAOS_PP_MOD_MACHINE.

Usage - C99 Specific 

CHAOS_PP_MOD_MACHINE_ID()

Remarks

This macro exists only for uniformity with other macros that require indirect invocation.

CHAOS_PP_MOD_MACHINE_

The CHAOS_PP_MOD_MACHINE_ macro expands to a λ expression representing CHAOS_PP_MOD_MACHINE.

Usage - C99 Specific 

CHAOS_PP_MOD_MACHINE_

Remarks

This macro is a shorthand form of the following λ binding: 

CHAOS_PP_LAMBDA(CHAOS_PP_MOD_MACHINE)
