CHAOS_PP_ARRAY_REVERSE

The CHAOS_PP_ARRAY_REVERSE macro reverses the elements in an array.

Usage 

CHAOS_PP_ARRAY_REVERSE(array)
array

The array whose elements are to be reversed. This array must contain CHAOS_PP_LIMIT_TUPLE or less elements.

Remarks

This macro expands to an array containing the elements of array in reverse order.

See Also

  • CHAOS_PP_SEQ_REVERSE_DUPLEX

Sample Code

#include <chaos/preprocessor/array/reverse.h>

#define DATA() (3, (a, b, c))

CHAOS_PP_ARRAY_REVERSE(DATA()) // (3, (c, b, a))

CHAOS_PP_ARRAY_REVERSE_ID

The CHAOS_PP_ARRAY_REVERSE_ID macro indirectly expands to CHAOS_PP_ARRAY_REVERSE.

Usage 

CHAOS_PP_ARRAY_REVERSE_ID()

Remarks

This macro effectively allows CHAOS_PP_ARRAY_REVERSE to pass through itself without becoming disabled.

CHAOS_PP_ARRAY_REVERSE_

The CHAOS_PP_ARRAY_REVERSE_ macro expands to a λ expression representing CHAOS_PP_ARRAY_REVERSE.

Usage - C99 Specific 

CHAOS_PP_ARRAY_REVERSE_

Remarks

This macro is a shorthand form of the following λ binding: 

CHAOS_PP_LAMBDA(CHAOS_PP_ARRAY_REVERSE_ID)()
