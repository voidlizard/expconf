CHAOS_PP_ARRAY_SIZE

The CHAOS_PP_ARRAY_SIZE macro extracts the size of an array.

Usage 

CHAOS_PP_ARRAY_SIZE(array)
array

The array whose size is to be extracted.

Remarks

This macro expands to the number of elements in array.

Sample Code

#include <chaos/preprocessor/array/size.h>

#define DATA() (3, (a, b, c))

CHAOS_PP_ARRAY_SIZE(DATA()) // 3

CHAOS_PP_ARRAY_SIZE_ID

The CHAOS_PP_ARRAY_SIZE_ID macro indirectly expands to CHAOS_PP_ARRAY_SIZE.

Usage 

CHAOS_PP_ARRAY_SIZE_ID()

Remarks

This macro exists only for uniformity with other macros that require indirect invocation.

CHAOS_PP_ARRAY_SIZE_

The CHAOS_PP_ARRAY_SIZE_ macro expands to a λ expression representing CHAOS_PP_ARRAY_SIZE.

Usage - C99 Specific 

CHAOS_PP_ARRAY_SIZE_

Remarks

This macro is a shorthand form of the following λ binding: 

CHAOS_PP_LAMBDA(CHAOS_PP_ARRAY_SIZE)
