The array/to_tuple.h header defines a macro that converts an array to a tuple.

Usage

#include <chaos/preprocessor/array/to_tuple.h>

Contents

  • CHAOS_PP_ARRAY_TO_TUPLE

CHAOS_PP_ARRAY_TO_TUPLE

The CHAOS_PP_ARRAY_TO_TUPLE macro converts an array to a tuple.

Usage 

CHAOS_PP_ARRAY_TO_TUPLE(array)
array

The array to be converted.

Remarks

This macro expands to a tuple containing the elements of array.

See Also

  • CHAOS_PP_ARRAY_TO_LIST
  • CHAOS_PP_ARRAY_TO_SEQ
  • CHAOS_PP_ARRAY_TO_STRING
  • CHAOS_PP_CAST

Sample Code

#include <chaos/preprocessor/array/to_tuple.h>

#define DATA() (3, (x, y, z))

CHAOS_PP_ARRAY_TO_TUPLE(DATA()) // (x, y, z)

CHAOS_PP_ARRAY_TO_TUPLE_ID

The CHAOS_PP_ARRAY_TO_TUPLE_ID macro indirectly expands to CHAOS_PP_ARRAY_TO_TUPLE.

Usage 

CHAOS_PP_ARRAY_TO_TUPLE_ID()

Remarks

This macro effectively allows CHAOS_PP_ARRAY_TO_TUPLE to pass through itself without becoming disabled.

CHAOS_PP_ARRAY_TO_TUPLE_

The CHAOS_PP_ARRAY_TO_TUPLE_ macro expands to a λ expression representing CHAOS_PP_ARRAY_TO_TUPLE.

Usage - C99 Specific 

CHAOS_PP_ARRAY_TO_TUPLE_

Remarks

This macro is a shorthand form of the following λ binding: 

CHAOS_PP_LAMBDA(CHAOS_PP_ARRAY_TO_TUPLE_ID)()
