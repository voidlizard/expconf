The control/placemarker_if.h header defines a macro that expands to an expression if a condition is a placemarker.

Usage

#include <chaos/preprocessor/control/placemarker_if.h>

Contents

  • CHAOS_PP_PLACEMARKER_IF

CHAOS_PP_PLACEMARKER_IF

The CHAOS_PP_PLACEMARKER_IF macro expands to its second argument if its first argument is a placemarker.

Usage - C99 Specific 

CHAOS_PP_PLACEMARKER_IF(p)(...)
[p]

The condition that determines whether the macro should expand to __VA_ARGS__ or nothing. Valid values are 0 and nothing (i.e. a placemarker).

[...]

The resulting expression if p is a placemarker.

Remarks

If p expands to 0, this macro expands to nothing. Otherwise, it expands to __VA_ARGS__.

This macro is lazy. If p is 0, __VA_ARGS__ is not evaluated.

Sample Code

#include <chaos/preprocessor/config.h>
#include <chaos/preprocessor/control/placemarker_if.h>

#if CHAOS_PP_VARIADICS

#define BRACKET(pseq) BRACKET_A pseq(0,)

#define BRACKET_A(p, ...) CHAOS_PP_PLACEMARKER_IF(p)([p ## __VA_ARGS__] BRACKET_B)
#define BRACKET_B(p, ...) CHAOS_PP_PLACEMARKER_IF(p)([p ## __VA_ARGS__] BRACKET_A)

BRACKET((,a)(,b)(,c)(,d)) // [a] [b] [c] [d]

#endif

CHAOS_PP_PLACEMARKER_IF_ID

The CHAOS_PP_PLACEMARKER_IF_ID macro indirectly expands to CHAOS_PP_PLACEMARKER_IF.

Usage - C99 Specific 

CHAOS_PP_PLACEMARKER_IF_ID()

Remarks

This macro exists only for uniformity with other macros that require indirect invocation.

CHAOS_PP_PLACEMARKER_IF_

The CHAOS_PP_PLACEMARKER_IF_ macro expands to a λ expression representing CHAOS_PP_PLACEMARKER_IF.

Usage - C99 Specific 

CHAOS_PP_PLACEMARKER_IF_

Remarks

This macro is a shorthand form of the following λ binding: 

CHAOS_PP_LAMBDA(CHAOS_PP_PLACEMARKER_IF)
