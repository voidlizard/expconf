The control/null.h header defines a macro that expands to a logically null header name.

Usage

#include <chaos/preprocessor/control/null.h>

Contents

  • CHAOS_PP_NULL

CHAOS_PP_NULL

The CHAOS_PP_NULL macro expands to a logically null header name.

Usage 

#include CHAOS_PP_NULL()

Remarks

Once the directive name of an #include directive is reached by the preprocessor, some header file must be included. This macro expands to a header file that has no effect on the translation unit (i.e. a header with no contents).
