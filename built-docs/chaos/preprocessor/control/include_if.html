The control/include_if.h header defines a macro used to conditionally include a header file.

Usage

#include <chaos/preprocessor/control/include_if.h>

Contents

  • CHAOS_PP_INCLUDE_IF

CHAOS_PP_INCLUDE_IF

The CHAOS_PP_INCLUDE_IF macro is used to conditionally include a header file.

Usage 

#include CHAOS_PP_INCLUDE_IF(cond, file)
cond

The condition that determines whether file should be included. Valid values range from 0 to CHAOS_PP_LIMIT_MAG.

file

The file to be included if cond is non-zero.

Remarks

If cond is non-zero, the file specified by file is included. Otherwise, CHAOS_PP_NULL() is included.

Note that the contents of a filename of the h-char-sequence form immediately become subject to macro expansion when macro expansion is used on an #include directive. For example, 

#define file other

#include <file.h>
#include CHAOS_PP_EMPTY() <file.h>

The first #include directive includes <file.h>, but the second includes <other.h>.

See Also

  • CHAOS_PP_NULL

Sample Code

#include <chaos/preprocessor/control/include_if.h>

#define TRACE 1 // externally defined

#include <iosfwd>
#include CHAOS_PP_INCLUDE_IF(TRACE, <iostream>)

CHAOS_PP_INCLUDE_IF_ID

The CHAOS_PP_INCLUDE_IF_ID macro indirectly expands to CHAOS_PP_INCLUDE_IF.

Usage 

CHAOS_PP_INCLUDE_IF_ID()

Remarks

This macro exists only for uniformity with other macros that require indirect invocation.

CHAOS_PP_INCLUDE_IF_

The CHAOS_PP_INCLUDE_IF_ macro expands to a λ expression representing CHAOS_PP_INCLUDE_IF.

Usage - C99 Specific 

CHAOS_PP_INCLUDE_IF_

Remarks

This macro is a shorthand form of the following λ binding: 

CHAOS_PP_LAMBDA(CHAOS_PP_INCLUDE_IF)
