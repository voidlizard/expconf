The string/to_upper.h header defines an algorithm that converts a string to uppercase.

Usage

#include <chaos/preprocessor/string/to_upper.h>

CHAOS_PP_TO_UPPER

The CHAOS_PP_TO_UPPER macro converts a character to uppercase.

Usage 

CHAOS_PP_TO_UPPER(c)
c

The character to be converted to uppercase. Valid values are the numerals (0 through 9), the alphabetic characters in the basic source character set (a through z and A through Z), and the underscore (_).

Remarks

If c is an lowercase letter, this macro expands to the letter's uppercase equivalent. Otherwise, it expands to c.

See Also

  • CHAOS_PP_TO_LOWER

Sample Code

#include <chaos/preprocessor/string/to_upper.h>

CHAOS_PP_TO_UPPER(a) // A
CHAOS_PP_TO_UPPER(A) // A
CHAOS_PP_TO_UPPER(0) // 0
CHAOS_PP_TO_UPPER(_) // _

CHAOS_PP_TO_UPPER_ID

The CHAOS_PP_TO_UPPER_ID macro indirectly expands to CHAOS_PP_TO_UPPER.

Usage 

CHAOS_PP_TO_UPPER_ID()

Remarks

This macro effectively allows CHAOS_PP_TO_UPPER to pass through itself without becoming disabled.

CHAOS_PP_TO_UPPER_

The CHAOS_PP_TO_UPPER_ macro expands to a λ expression representing CHAOS_PP_TO_UPPER.

Usage - C99 Specific 

CHAOS_PP_TO_UPPER_

Remarks

This macro is a shorthand form of the following λ binding: 

CHAOS_PP_LAMBDA(CHAOS_PP_TO_UPPER_ID)()

CHAOS_PP_STRING_TO_UPPER

The CHAOS_PP_STRING_TO_UPPER macro converts its argument to uppercase.

Usage - C99 Specific 

CHAOS_PP_STRING_TO_UPPER(string)
[string]

The string to be converted to uppercase.

Usage - C/C++ Specific 

CHAOS_PP_STRING_TO_UPPER(string)
string

The string to be converted to uppercase.

Remarks

This macro expands to a string containing the elements of string converted to uppercase. For example, 

CHAOS_PP_STRING_TO_UPPER(c h a o s) // C H A O S

This macro uses bypass semantics. If bypass semantics are already in effect, CHAOS_PP_STRING_TO_UPPER_BYPASS must be used instead.

See Also

  • CHAOS_PP_STRING_TO_LOWER

Sample Code

#include <chaos/preprocessor/algorithm/concat.h>
#include <chaos/preprocessor/string/core.h>
#include <chaos/preprocessor/string/to_upper.h>

CHAOS_PP_CONCAT(
    (CHAOS_PP_STRING) CHAOS_PP_STRING_TO_UPPER(
        a b C _ 1 0
    )
)
// ABC_10

CHAOS_PP_STRING_TO_UPPER_BYPASS

The CHAOS_PP_STRING_TO_UPPER_BYPASS macro behaves identically to CHAOS_PP_STRING_TO_UPPER except that it is parametized by the bypass recursion state.

Usage - C99 Specific 

CHAOS_PP_STRING_TO_UPPER_BYPASS(s, string)
s

The current bypass recursion state.

[string]

The string to be converted to uppercase.

Usage - C/C++ Specific 

CHAOS_PP_STRING_TO_UPPER_BYPASS(s, string)
s

The current bypass recursion state.

string

The string to be converted to uppercase.

Remarks

If bypass semantics are already in effect, this macro must be used instead of CHAOS_PP_STRING_TO_UPPER.
