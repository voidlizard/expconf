CHAOS_PP_STRING

The CHAOS_PP_STRING macro designates the name of the string data structure.

Usage 

CHAOS_PP_STRING

Remarks

This macro expands to CHAOS_PP_STRING. In other words, the identifier CHAOS_PP_STRING is defined as itself.

CHAOS_PP_STRING_NIL

The CHAOS_PP_STRING_NIL macro expands to a nil string.

Usage 

CHAOS_PP_STRING_NIL()

Remarks

In C99, this macro expands to nothing. (More generally, a nil string is an empty sequence of preprocessing tokens.) In C++ (and C prior to C99), there is no such thing as a nil string because "nothing" cannot be passed as an argument. Hence, in C++ (and C prior to C99), this macro expands to a rogue value (that is uniquely recognized by the other macros in this header) such that generic algorithms can behave as if nil strings exist.

See Also

  • CHAOS_PP_ARRAY_NIL
  • CHAOS_PP_LIST_NIL
  • CHAOS_PP_NIL
  • CHAOS_PP_SEQ_NIL
  • CHAOS_PP_TUPLE_NIL
  • CHAOS_PP_VARIADIC_SEQ_NIL

Sample Code

#include <chaos/preprocessor/config.h>
#include <chaos/preprocessor/string/core.h>

#if CHAOS_PP_VARIADICS

CHAOS_PP_STRING_NIL() //

#else

CHAOS_PP_STRING_NIL() // unspecified

#endif

CHAOS_PP_STRING_CONS(
    CHAOS_PP_STRING_CONS(
        CHAOS_PP_STRING_NIL(),
        b
    ),
    a
)
// a

CHAOS_PP_STRING_CONS

The CHAOS_PP_STRING_CONS macro appends an element to the head of a string.

Usage - C99 Specific 

CHAOS_PP_STRING_CONS(string, x)
[string]

The string to which x is to be appended.

x

The element to be appended to the head of string.

Usage - C/C++ Specific 

CHAOS_PP_STRING_CONS(string, x)
string

The string to which x is to be appended.

x

The element to be appended to the head of string.

Remarks

This macro expands to a string containing the elements of string with x appended to the head.

See Also

  • CHAOS_PP_ARRAY_CONS
  • CHAOS_PP_CONS
  • CHAOS_PP_LIST_CONS
  • CHAOS_PP_SEQ_CONS
  • CHAOS_PP_TUPLE_CONS
  • CHAOS_PP_VARIADIC_SEQ_CONS

Sample Code

#include <chaos/preprocessor/string/core.h>

CHAOS_PP_STRING_CONS(
    CHAOS_PP_STRING_CONS(
        CHAOS_PP_STRING_CONS(
            CHAOS_PP_STRING_NIL(), z
        ), y
    ), x
)
// x y z

CHAOS_PP_STRING_ITEM

The CHAOS_PP_STRING_ITEM macro is an identity macro used to encapsulate elements.

Usage 

CHAOS_PP_STRING_ITEM(x)
x

The element of a string to be encapsulated.

Remarks

This macro is used by generic algorithms to encapsulate elements before they are passed to user-defined macros. Elements in a string cannot be variadic. Therefore, this macro is a simple identity macro.

CHAOS_PP_STRING_IS_CONS

The CHAOS_PP_STRING_IS_CONS macro is a predicate that tests whether a string is non-nil.

Usage - C99 Specific 

CHAOS_PP_STRING_IS_CONS(string)
[string]

The string to be tested.

Usage - C/C++ Specific 

CHAOS_PP_STRING_IS_CONS(string)
string

The string to be tested.

Remarks

If string is non-nil, this macro expands to 1. Otherwise, it expands to 0.

See Also
