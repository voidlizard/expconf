The punctuation/comma.h header defines a macro that expands to a comma.

Usage

#include <chaos/preprocessor/punctuation/comma.h>

Contents

  • CHAOS_PP_COMMA

CHAOS_PP_COMMA

The CHAOS_PP_COMMA macro expands to a comma.

Usage 

CHAOS_PP_COMMA()

Remarks

When invoked, this macro expands to a comma preprocessing token.

See Also

  • CHAOS_PP_COMMA_IF

Sample Code

#include <chaos/preprocessor/punctuation/comma.h>

#define MACRO(p) a p() b

MACRO(CHAOS_PP_COMMA) // a, b

CHAOS_PP_COMMA_ID

The CHAOS_PP_COMMA_ID macro indirectly expands to CHAOS_PP_COMMA.

Usage 

CHAOS_PP_COMMA_ID()

Remarks

This macro exists only for uniformity with other macros that require indirect invocation.

CHAOS_PP_COMMA_

The CHAOS_PP_COMMA_ macro expands to a λ expression representing CHAOS_PP_COMMA.

Usage - C99 Specific 

CHAOS_PP_COMMA_

Remarks

This macro is a shorthand form of the following λ binding: 

CHAOS_PP_LAMBDA(CHAOS_PP_COMMA)
