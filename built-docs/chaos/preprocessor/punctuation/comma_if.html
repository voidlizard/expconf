The punctuation/comma_if.h header defines a macro that conditionally expands to a comma.

Usage

#include <chaos/preprocessor/punctuation/comma_if.h>

Contents

  • CHAOS_PP_COMMA_IF

CHAOS_PP_COMMA_IF

The CHAOS_PP_COMMA_IF macro expands to a comma if its argument is non-zero.

Usage 

CHAOS_PP_COMMA_IF(cond)
cond

The condition that determines whether the macro should expand to a comma. Valid values range from 0 to CHAOS_PP_LIMIT_MAG.

Remarks

If cond is non-zero, this macro expands to a comma preprocessing token. Otherwise, it expands to nothing.

See Also

  • CHAOS_PP_COMMA

Sample Code

#include <chaos/preprocessor/punctuation/comma_if.h>
#include <chaos/preprocessor/recursion/expr.h>
#include <chaos/preprocessor/repetition/repeat.h>

#define MACRO(s, n, _) \
    CHAOS_PP_COMMA_IF(n) class T ## n \
    /**/

CHAOS_PP_EXPR(CHAOS_PP_REPEAT(
    3, MACRO, ~
))
// class T0, class T1, class T2

CHAOS_PP_COMMA_IF_ID

The CHAOS_PP_COMMA_IF_ID macro indirectly expands to CHAOS_PP_COMMA_IF.

Usage 

CHAOS_PP_COMMA_IF_ID()

Remarks

This macro exists only for uniformity with other macros that require indirect invocation.

CHAOS_PP_COMMA_IF_

The CHAOS_PP_COMMA_IF_ macro expands to a λ expression representing CHAOS_PP_COMMA_IF.

Usage - C99 Specific 

CHAOS_PP_COMMA_IF_

Remarks

This macro is a shorthand form of the following λ binding: 

CHAOS_PP_LAMBDA(CHAOS_PP_COMMA_IF)
