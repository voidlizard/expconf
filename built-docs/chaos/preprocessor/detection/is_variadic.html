The detection/is_variadic.h header defines a macro used to detect parenthetic expressions.

Usage

#include <chaos/preprocessor/detection/is_variadic.h>

Contents

  • CHAOS_PP_IS_VARIADIC

CHAOS_PP_IS_VARIADIC

The CHAOS_PP_IS_VARIADIC macro detects whether its argument begins with a parenthetic expression of any arity.

Usage - C99 Specific 

CHAOS_PP_IS_VARIADIC(...)
[...]

The expression to be tested.

Remarks

If x or __VA_ARGS__ begins with a parenthetic expression, this macro expands to 1. Otherwise, it expands to 0.

See Also

  • CHAOS_PP_IS_BINARY
  • CHAOS_PP_IS_NULLARY
  • CHAOS_PP_IS_TERNARY
  • CHAOS_PP_IS_UNARY

Sample Code

#include <chaos/preprocessor/config.h>
#include <chaos/preprocessor/detection/is_variadic.h>

#if CHAOS_PP_VARIADICS

CHAOS_PP_IS_VARIADIC(++) // 0

CHAOS_PP_IS_VARIADIC(())     // 1
CHAOS_PP_IS_VARIADIC((a))    // 1
CHAOS_PP_IS_VARIADIC((a, b)) // 1

#endif

CHAOS_PP_IS_VARIADIC_ID

The CHAOS_PP_IS_VARIADIC_ID macro indirectly expands to CHAOS_PP_IS_VARIADIC.

Usage - C99 Specific 

CHAOS_PP_IS_VARIADIC_ID()

Remarks

This macro exists only for uniformity with other macros that require indirect invocation.

CHAOS_PP_IS_VARIADIC_

The CHAOS_PP_IS_VARIADIC_ macro expands to a λ expression representing CHAOS_PP_IS_VARIADIC.

Usage - C99 Specific 

CHAOS_PP_IS_VARIADIC_

Remarks

This macro is a shorthand form of the following λ binding: 

CHAOS_PP_LAMBDA(CHAOS_PP_IS_VARIADIC)
