The detection/is_ternary.h header defines a macro used to detect ternary parenthetic expressions.

Usage

#include <chaos/preprocessor/detection/is_ternary.h>

Contents

  • CHAOS_PP_IS_TERNARY

CHAOS_PP_IS_TERNARY

The CHAOS_PP_IS_TERNARY macro detects whether its argument begins with a ternary parenthetic expression.

Usage - C99 Specific 

CHAOS_PP_IS_TERNARY(...)
[...]

The expression to be tested.

Usage - C/C++ Specific 

CHAOS_PP_IS_TERNARY(x)
x

The expression to be tested.

Remarks

If x or __VA_ARGS__ begins with a ternary parenthetic expression, this macro expands to 1. Otherwise, it expands to 0.

The expression to be tested must begin with a ternary parenthetic expression or a non-parenthetic expression. A parenthetic expression of any other arity is invalid input.

See Also

  • CHAOS_PP_IS_BINARY
  • CHAOS_PP_IS_NULLARY
  • CHAOS_PP_IS_UNARY
  • CHAOS_PP_IS_VARIADIC

Sample Code

#include <chaos/preprocessor/detection/is_ternary.h>

CHAOS_PP_IS_TERNARY(++)        // 0
CHAOS_PP_IS_TERNARY((a, b, c)) // 1

CHAOS_PP_IS_TERNARY_ID

The CHAOS_PP_IS_TERNARY_ID macro indirectly expands to CHAOS_PP_IS_TERNARY.

Usage 

CHAOS_PP_IS_TERNARY_ID()

Remarks

This macro exists only for uniformity with other macros that require indirect invocation.

CHAOS_PP_IS_TERNARY_

The CHAOS_PP_IS_TERNARY_ macro expands to a λ expression representing CHAOS_PP_IS_TERNARY.

Usage - C99 Specific 

CHAOS_PP_IS_TERNARY_

Remarks

This macro is a shorthand form of the following λ binding: 

CHAOS_PP_LAMBDA(CHAOS_PP_IS_TERNARY)
