The limits.h header defines macros that specify the physical limits of various library facilities.

Usage

#include <chaos/preprocessor/limits.h>

CHAOS_PP_LIMIT_AS_ARGS

The CHAOS_PP_LIMIT_AS_ARGS macro expands to the maximum arity that is transformable by CHAOS_PP_AS_ARGS.

Usage 

CHAOS_PP_LIMIT_AS_ARGS

Remarks

This macro currently expands to 5.

CHAOS_PP_LIMIT_AS_ARGS_ID

The CHAOS_PP_LIMIT_AS_ARGS_ID macro indirectly expands to CHAOS_PP_LIMIT_AS_ARGS.

Usage 

CHAOS_PP_LIMIT_AS_ARGS_ID()

Remarks

This macro effectively allows CHAOS_PP_LIMIT_AS_ARGS to be passed around without expansion.

CHAOS_PP_LIMIT_AS_ARGS_

The CHAOS_PP_LIMIT_AS_ARGS_ macro expands to a 位 expression representing CHAOS_PP_LIMIT_AS_ARGS.

Usage - C99 Specific 

CHAOS_PP_LIMIT_AS_ARGS_

Remarks

This macro is a shorthand form of the following 位 binding: 

CHAOS_PP_LAMBDA(CHAOS_PP_LIMIT_AS_ARGS_ID)()
