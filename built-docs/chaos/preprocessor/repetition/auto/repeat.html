The repetition/auto/repeat.h header defines an algorithm that repeats a user-specified target a specified number of times.

Usage

#include <chaos/preprocessor/repetition/auto/repeat.h>

Contents

  • CHAOS_PP_AUTO_REPEAT

CHAOS_PP_AUTO_REPEAT

The CHAOS_PP_AUTO_REPEAT macro repeats its second argument the number of times specified by its first argument.

Usage - C99 Specific 

CHAOS_PP_AUTO_REPEAT(count, macro)
count

The number of times that macro is to be repeated. Valid values range from 0 to CHAOS_PP_LIMIT_MAG.

macro

The target that is to be repeated. This target must be either a λ expression or a macro name that refers to a macro with a signature compatible with the following pseudo-signature: 

macro(s, n)

Usage - C99 Specific 

CHAOS_PP_AUTO_REPEAT(count, macro, ...)
count

The number of times that macro is to be repeated. Valid values range from 0 to CHAOS_PP_LIMIT_MAG.

macro

The target that is to be repeated. This target must be either a λ expression or a macro name that refers to a macro with a signature compatible with the following pseudo-signature: 

macro(s, n, data†)

† The data pseudo-parameter must be either ... or the number of parameters that corresponds to the number of arguments in the auxiliary data, __VA_ARGS__.

[...]

Auxiliary data to be passed through the algorithm to the target.

Remarks

The CHAOS_PP_AUTO_REPEAT macro is a higher-order thunk that deduces the current higher-order entry point and invokes CHAOS_PP_REPEAT.

This macro may not be used if bypass semantics are in effect.

Sample Code

#include <chaos/preprocessor/config.h>
#include <chaos/preprocessor/repetition/auto/repeat.h>

#if CHAOS_PP_VARIADICS

#define MACRO(s, n) var[n] = n;

CHAOS_PP_AUTO_REPEAT(3, MACRO) // var[0] = 0; var[1] = 1; var[2] = 2;

#endif

CHAOS_PP_AUTO_REPEAT_ID

The CHAOS_PP_AUTO_REPEAT_ID macro indirectly expands to CHAOS_PP_AUTO_REPEAT.

Usage - C99 Specific 

CHAOS_PP_AUTO_REPEAT_ID()

Remarks

This macro effectively allows CHAOS_PP_AUTO_REPEAT to be passed around without expansion.

CHAOS_PP_AUTO_REPEAT_

The CHAOS_PP_AUTO_REPEAT_ macro expands to a λ expression representing CHAOS_PP_AUTO_REPEAT.

Usage - C99 Specific 

CHAOS_PP_AUTO_REPEAT_

Remarks

This macro is a shorthand form of the following λ binding: 

CHAOS_PP_LAMBDA(CHAOS_PP_AUTO_REPEAT_ID)()
