The repetition/auto/enum_shifted_trailing.h header defines an algorithm that enumerates a user-specified target a specified number of times.

Usage

#include <chaos/preprocessor/repetition/auto/enum_shifted_trailing.h>

Contents

CHAOS_PP_AUTO_ENUM_SHIFTED_TRAILING

The CHAOS_PP_AUTO_ENUM_SHIFTED_TRAILING macro enumerates its second argument one less time than the number specified by its first argument.

Usage - C99 Specific 

CHAOS_PP_AUTO_ENUM_SHIFTED_TRAILING(count, macro)
count

One greater than the number of times that macro is to be repeated. Valid values range from 0 to CHAOS_PP_LIMIT_MAG.

macro

The target that is to be repeated. This target must be either a λ expression or a macro name that refers to a macro with a signature compatible with the following pseudo-signature: 

macro(s, n)

Usage - C99 Specific 

CHAOS_PP_AUTO_ENUM_SHIFTED_TRAILING(count, macro, ...)
count

One greater than the number of times that macro is to be repeated. Valid values range from 0 to CHAOS_PP_LIMIT_MAG.

macro

The target that is to be repeated. This target must be either a λ expression or a macro name that refers to a macro with a signature compatible with the following pseudo-signature: 

macro(s, n, data†)

† The data pseudo-parameter must be either ... or the number of parameters that corresponds to the number of arguments in the auxiliary data, __VA_ARGS__.

[...]

Auxiliary data to be passed through the algorithm to the target.

Remarks

The CHAOS_PP_AUTO_ENUM_SHIFTED_TRAILING macro is a higher-order thunk that deduces the current higher-order entry point and invokes CHAOS_PP_ENUM_SHIFTED_TRAILING.

This macro may not be used if bypass semantics are in effect.

CHAOS_PP_AUTO_ENUM_SHIFTED_TRAILING_ID

The CHAOS_PP_AUTO_ENUM_SHIFTED_TRAILING_ID macro indirectly expands to CHAOS_PP_AUTO_ENUM_SHIFTED_TRAILING.

Usage - C99 Specific 

CHAOS_PP_AUTO_ENUM_SHIFTED_TRAILING_ID()

Remarks

This macro effectively allows CHAOS_PP_AUTO_ENUM_SHIFTED_TRAILING to be passed around without expansion.

CHAOS_PP_AUTO_ENUM_SHIFTED_TRAILING_

The CHAOS_PP_AUTO_ENUM_SHIFTED_TRAILING_ macro expands to a λ expression representing CHAOS_PP_AUTO_ENUM_SHIFTED_TRAILING.

Usage - C99 Specific 

CHAOS_PP_AUTO_ENUM_SHIFTED_TRAILING_

Remarks

This macro is a shorthand form of the following λ binding: 

CHAOS_PP_LAMBDA(CHAOS_PP_AUTO_ENUM_SHIFTED_TRAILING_ID)()
