The list/core.h header defines the core list operations necessary to use a list as a generic data structure.

CHAOS_PP_LIST

The CHAOS_PP_LIST macro designates the name of the list data structure.

Usage 

CHAOS_PP_LIST

Remarks

This macro expands to CHAOS_PP_LIST. In other words, the identifier CHAOS_PP_LIST is defined as itself.

See Also

  • CHAOS_PP_SEQ
  • CHAOS_PP_STRING
  • CHAOS_PP_TUPLE
  • CHAOS_PP_VARIADIC_SEQ

CHAOS_PP_LIST_ID

The CHAOS_PP_LIST_ID macro indirectly expands to CHAOS_PP_LIST.

Usage 

CHAOS_PP_LIST_ID()

Remarks

This macro exists only for uniformity with other macros that require indirect invocation.

CHAOS_PP_LIST_

The CHAOS_PP_LIST_ macro expands to a 位 expression representing CHAOS_PP_LIST.

Usage - C99 Specific 

CHAOS_PP_LIST_

Remarks

This macro is a shorthand form of the following 位 binding: 

CHAOS_PP_LAMBDA(CHAOS_PP_LIST)

CHAOS_PP_LIST_TAG

The CHAOS_PP_LIST_TAG macro is an identity macro used to compare type names.

Usage 

CHAOS_PP_LIST_TAG(tag)
tag

The tag used to distinguish between data structures.

Remarks

This macro is used by generic algorithms to compare the types of generic data structures. It should be used for no other purpose.

CHAOS_PP_LIST_CONS

The CHAOS_PP_LIST_CONS macro appends an element to the head of a list.

Usage - C99 Specific 

CHAOS_PP_LIST_CONS(list, x)
list

The list to which x is to be appended.

[x]

The element to be appended to the head of list.

Usage - C/C++ Specific 

CHAOS_PP_LIST_CONS(list, x)
list

The list to which x is to be appended.

x

The element to be appended to the head of list.

Remarks

This macro expands to a list containing the elements of list with x appended to the head.

See Also

  • CHAOS_PP_ARRAY_CONS
  • CHAOS_PP_CONS
  • CHAOS_PP_SEQ_CONS
  • CHAOS_PP_STRING_CONS
  • CHAOS_PP_TUPLE_CONS
  • CHAOS_PP_VARIADIC_SEQ_CONS

Sample Code

#include <chaos/preprocessor/list/core.h>

#define DATA() (b, (c, ...))

CHAOS_PP_LIST_CONS(DATA(), a) // (a, (b, (c, ...)))

CHAOS_PP_LIST_ITEM

The CHAOS_PP_LIST_ITEM macro is an identity macro used to encapsulate elements.

Usage - C99 Specific 

CHAOS_PP_LIST_ITEM(x)
[x]

The element of a list to be encapsulated.

Usage - C/C++ Specific 

CHAOS_PP_LIST_ITEM(x)
x

The element of a list to be encapsulated.

Remarks

This macro is used by generic algorithms to encapsulate elements before they are passed to user-defined macros. Elements in a list cannot be variadic. Therefore, this macro is a simple identity macro.

CHAOS_PP_LIST_IS_CONS

The CHAOS_PP_LIST_IS_CONS macro is a predicate that tests whether a list is non-nil.

Usage 

CHAOS_PP_LIST_IS_CONS(list)
list

The list to be tested.

Remarks

If list is non-nil, this macro expands to 1. Otherwise, it expands to 0.

See Also

  • CHAOS_PP_ARRAY_IS_CONS
  • CHAOS_PP_IS_CONS
  • CHAOS_PP_LIST_IS_NIL
  • CHAOS_PP_SEQ_IS_CONS
  • CHAOS_PP_STRING_IS_CONS
  • CHAOS_PP_TUPLE_IS_CONS
  • CHAOS_PP_VARIADIC_SEQ_IS_CONS

Sample Code

#include <chaos/preprocessor/list/core.h>

CHAOS_PP_LIST_IS_CONS(...)           // 0
