The comparison/equal.h header defines an equality operator.

Usage

#include <chaos/preprocessor/comparison/equal.h>

Contents

  • CHAOS_PP_EQUAL

CHAOS_PP_EQUAL

The CHAOS_PP_EQUAL macro compares two values for equality.

Usage 

CHAOS_PP_EQUAL(x, y)
x

The first operand. Valid values range from 0 to CHAOS_PP_LIMIT_MAG.

y

The second operand. Valid values range from 0 to CHAOS_PP_LIMIT_MAG.

Remarks

If x is equal to y, this macro expands to 1. Otherwise, it expands to 0.

See Also

  • CHAOS_PP_ARBITRARY_EQUAL

Sample Code

#include <chaos/preprocessor/comparison/equal.h>

CHAOS_PP_EQUAL(12, 34) // 0
CHAOS_PP_EQUAL(56, 56) // 1

CHAOS_PP_EQUAL_ID

The CHAOS_PP_EQUAL_ID macro indirectly expands to CHAOS_PP_EQUAL.

Usage 

CHAOS_PP_EQUAL_ID()

Remarks

This macro exists only for uniformity with other macros that require indirect invocation.

CHAOS_PP_EQUAL_

The CHAOS_PP_EQUAL_ macro expands to a λ expression representing CHAOS_PP_EQUAL.

Usage - C99 Specific 

CHAOS_PP_EQUAL_

Remarks

This macro is a shorthand form of the following λ binding: 

CHAOS_PP_LAMBDA(CHAOS_PP_EQUAL)
