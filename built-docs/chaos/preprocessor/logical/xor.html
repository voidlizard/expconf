Usage

#include <chaos/preprocessor/logical/xor.h>

Contents

CHAOS_PP_XOR

The CHAOS_PP_XOR macro expands to the logical XOR of its operands.

Usage 

CHAOS_PP_XOR(p)(q)
p

The left operand. Valid values range from 0 to CHAOS_PP_LIMIT_MAG.

q

The right operand. Valid values range from 0 to CHAOS_PP_LIMIT_MAG.

Remarks

If either p or q is non-zero exclusively, this macro expands to 1. Otherwise, it expands to 0.

This macro performs a Boolean conversion on each operand. If this conversion is undesirable, CHAOS_PP_BITXOR may be used instead.

See Also

  • CHAOS_PP_BITXOR

Sample Code

#include <chaos/preprocessor/logical/xor.h>

CHAOS_PP_XOR(4)(7) // 0
CHAOS_PP_XOR(5)(0) // 1

CHAOS_PP_XOR_ID

The CHAOS_PP_XOR_ID macro indirectly expands to CHAOS_PP_XOR.

Usage 

CHAOS_PP_XOR_ID()

Remarks

This macro exists only for uniformity with other macros that require indirect invocation.

CHAOS_PP_XOR_

The CHAOS_PP_XOR_ macro expands to a λ expression representing CHAOS_PP_XOR.

Usage - C99 Specific 

CHAOS_PP_XOR_

Remarks

This macro is a shorthand form of the following λ binding: 

CHAOS_PP_LAMBDA(CHAOS_PP_XOR)
