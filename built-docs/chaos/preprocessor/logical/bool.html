The logical/bool.h header defines a macro that performs a Boolean conversion on an integer literal.

Usage

#include <chaos/preprocessor/logical/bool.h>

Contents

  • CHAOS_PP_BOOL

CHAOS_PP_BOOL

The CHAOS_PP_BOOL macro performs a Boolean conversion on its argument.

Usage 

CHAOS_PP_BOOL(x)
x

The integer literal to be converted. Valid values range from 0 to CHAOS_PP_LIMIT_MAG.

Remarks

If x expands to 0, this macro expands to 0. Otherwise, it expands to 1.

See Also

  • CHAOS_PP_ARBITRARY_BOOL

Sample Code

#include <chaos/preprocessor/logical/bool.h>

CHAOS_PP_BOOL(100) // 1
CHAOS_PP_BOOL(0)   // 0

CHAOS_PP_BOOL_ID

The CHAOS_PP_BOOL_ID macro indirectly expands to CHAOS_PP_BOOL.

Usage 

CHAOS_PP_BOOL_ID()

Remarks

This macro exists only for uniformity with other macros that require indirect invocation.

CHAOS_PP_BOOL_

The CHAOS_PP_BOOL_ macro expands to a λ expression representing CHAOS_PP_BOOL.

Usage - C99 Specific 

CHAOS_PP_BOOL_

Remarks

This macro is a shorthand form of the following λ binding: 

CHAOS_PP_LAMBDA(CHAOS_PP_BOOL)
