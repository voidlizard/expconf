The lambda/execute.h header defines IL operators and a macro that executes IL expressions.

Usage

#include <chaos/preprocessor/lambda/execute.h>

Contents

  • CHAOS_PP_ELEMENT
  • CHAOS_PP_EXECUTE
  • CHAOS_PP_OPERATOR

CHAOS_PP_ELEMENT

The CHAOS_PP_ELEMENT macro is an IL operator that produces an IL statement representing arbitrary preprocessing tokens.

Usage - C99 Specific 

CHAOS_PP_ELEMENT(...)
[...]

The preprocessing tokens to be represented as an IL statement.

Usage - C/C++ Specific 

CHAOS_PP_ELEMENT(x)
x

The preprocessing tokens to be represented as an IL statement.

Remarks

This macro produces an IL statement representing x or __VA_ARGS__.

See Also

  • CHAOS_PP_EXECUTE
  • CHAOS_PP_OPERATOR

Sample Code

#include <chaos/preprocessor/lambda/execute.h>

CHAOS_PP_EXECUTE(
    CHAOS_PP_ELEMENT(x) CHAOS_PP_ELEMENT(+) CHAOS_PP_ELEMENT(y)
)
// x + y

CHAOS_PP_ELEMENT_ID

The CHAOS_PP_ELEMENT_ID macro indirectly expands to CHAOS_PP_ELEMENT.

Usage 

CHAOS_PP_ELEMENT_ID()

Remarks

This macro effectively allows CHAOS_PP_ELEMENT to pass through itself without becoming disabled.

CHAOS_PP_ELEMENT_

The CHAOS_PP_ELEMENT_ macro expands to a λ expression representing CHAOS_PP_ELEMENT.

Usage - C99 Specific 

CHAOS_PP_ELEMENT_

Remarks

This macro is a shorthand form of the following λ binding: 

CHAOS_PP_LAMBDA(CHAOS_PP_ELEMENT_ID)()

CHAOS_PP_OPERATOR

The CHAOS_PP_OPERATOR macro is an IL operator that produces an IL statement representing a nullary operator.

Usage 

CHAOS_PP_OPERATOR(id)
id

The operator to be represented as an IL statement. This operator must be a macro name that refers to a macro with a signature compatible with the following pseudo-signature: 

id()

Remarks

This macro produces an IL statement representing an invocation of id. It is used primarily to encode preprocessing tokens that are operative (i.e. those that interfere with macro expansion because they have syntactic meaning, such as commas and parentheses).

See Also

  • CHAOS_PP_ELEMENT
  • CHAOS_PP_EXECUTE

Sample Code

#include <chaos/preprocessor/lambda/execute.h>
#include <chaos/preprocessor/punctuation/comma.h>
#include <chaos/preprocessor/punctuation/paren.h>

CHAOS_PP_EXECUTE(
    CHAOS_PP_OPERATOR(CHAOS_PP_RPAREN)
        CHAOS_PP_OPERATOR(CHAOS_PP_COMMA)
            CHAOS_PP_OPERATOR(CHAOS_PP_LPAREN)
)
// ) , (

CHAOS_PP_EXECUTE

The CHAOS_PP_EXECUTE macro executes an IL expression.

Usage - C99 Specific 

CHAOS_PP_EXECUTE(il)
[il]

The IL expression to be executed.

Usage - C/C++ Specific 

CHAOS_PP_EXECUTE(il)
il

The IL expression to be executed.

Remarks

This macro produces the deferred expression represented by il.

See Also

  • CHAOS_PP_ELEMENT
  • CHAOS_PP_OPERATOR

Sample Code

#include <chaos/preprocessor/cat.h>
#include <chaos/preprocessor/lambda/execute.h>
#include <chaos/preprocessor/punctuation/comma.h>
#include <chaos/preprocessor/punctuation/paren.h>
#include <chaos/preprocessor/recursion/expr.h>

#define IL \
    CHAOS_PP_ELEMENT(CHAOS_PP_CAT_ID) \
    CHAOS_PP_ELEMENT(()) \
    CHAOS_PP_OPERATOR(CHAOS_PP_LPAREN) \
    CHAOS_PP_ELEMENT(1) \
    CHAOS_PP_OPERATOR(CHAOS_PP_COMMA) \
    CHAOS_PP_ELEMENT(2) \
    CHAOS_PP_OPERATOR(CHAOS_PP_RPAREN) \
    /**/

CHAOS_PP_EXECUTE(IL) // CHAOS_PP_CAT_ID()(1, 2)

CHAOS_PP_EXPR(
    CHAOS_PP_EXECUTE(IL)
)
// 12
