The tuple/reverse.h header defines macros that reverse the elements in a tuple.

Usage

#include <chaos/preprocessor/tuple/reverse.h>

CHAOS_PP_TUPLE_REVERSE

The CHAOS_PP_TUPLE_REVERSE macro reverses the elements in a tuple of the specified size.

Usage 

CHAOS_PP_TUPLE_REVERSE(size, tuple)
size

The size of the tuple whose elements are to be reversed. Valid values range from 0 to CHAOS_PP_LIMIT_TUPLE.

tuple

The tuple whose elements are to be reversed.

Remarks

This macro expands to a tuple containing the elements of tuple in reverse order.

Sample Code

CHAOS_PP_TUPLE_REVERSE(3, (a, b, c)) // (c, b, a)

CHAOS_PP_TUPLE_REVERSE_ID

The CHAOS_PP_TUPLE_REVERSE_ID macro indirectly expands to CHAOS_PP_TUPLE_REVERSE.

Usage 

CHAOS_PP_TUPLE_REVERSE_ID()

Remarks

This macro effectively allows CHAOS_PP_TUPLE_REVERSE to pass through itself without becoming disabled.

CHAOS_PP_TUPLE_REVERSE_

The CHAOS_PP_TUPLE_REVERSE_ macro expands to a λ expression representing CHAOS_PP_TUPLE_REVERSE.

Usage - C99 Specific 

CHAOS_PP_TUPLE_REVERSE_

Remarks

This macro is a shorthand form of the following λ binding: 

CHAOS_PP_LAMBDA(CHAOS_PP_TUPLE_REVERSE_ID)()

CHAOS_PP_TUPLE_REVERSE_ALT

The CHAOS_PP_TUPLE_REVERSE macro reverses the elements in its argument.

Usage - C99 Specific 

CHAOS_PP_TUPLE_REVERSE_ALT(tuple)
[tuple]

The tuple whose elements are to be reversed.

Remarks

This macro expands to a tuple containing the elements of tuple in reverse order.

The efficiency of the algorithm's use of the recursion backend can be described in terms of the number of recursion steps required to complete the algorithm. The asymptotic upper and lower bounds are Θ, where n is the number of elements in tuple and tω is CHAOS_PP_LIMIT_TUPLE.

This macro operates on the nil tuple representation used by the core tuple operations defined in <chaos/preprocessor/tuple/core.h>. Specifically, () is a one-element tuple, not a zero-element tuple.

This macro uses bypass semantics. If bypass semantics are already in effect, CHAOS_PP_TUPLE_REVERSE_ALT_BYPASS must be used instead.

CHAOS_PP_TUPLE_REVERSE_ALT_BYPASS

The CHAOS_PP_TUPLE_REVERSE_ALT_BYPASS macro behaves identically to CHAOS_PP_TUPLE_REVERSE_ALT except that it is parametized by the bypass recursion state.

Usage - C99 Specific 

CHAOS_PP_TUPLE_REVERSE_ALT_BYPASS(s, tuple)
s

The current bypass recursion state.

[tuple]

The tuple whose elements are to be reversed.

Remarks

If bypass semantics are already in effect, this macro must be used instead of CHAOS_PP_TUPLE_REVERSE_ALT.
