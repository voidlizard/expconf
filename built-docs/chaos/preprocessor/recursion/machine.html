CHAOS_PP_MACHINE

The CHAOS_PP_MACHINE macro invokes the continuation machine.

Usage - C99 Specific 

CHAOS_PP_EXPR(CHAOS_PP_MACHINE(k))
k

The continuation to be executed.

Remarks

This macro executes the continuation specified by k.

This macro may not be used inside a continuation machine instruction.

Sample Code

#include <chaos/preprocessor/arithmetic/dec.h>
#include <chaos/preprocessor/arithmetic/inc.h>
#include <chaos/preprocessor/config.h>
#include <chaos/preprocessor/control/if.h>
#include <chaos/preprocessor/recursion/expr.h>
#include <chaos/preprocessor/recursion/machine.h>

#if CHAOS_PP_VARIADICS

#define CHAOS_PP_INSTRUCTION_add(p, x, y, ...) \
    CHAOS_PP_IF(y)( \
        (, CHAOS_PP_INC(x), (add), CHAOS_PP_DEC(y), p ## __VA_ARGS__), \
        (, x, p ## __VA_ARGS__) \
    ) \
    /**/

#define ADD(x, y) \
    CHAOS_PP_EXPR(CHAOS_PP_MACHINE( \
        (, x, (add), y, (0xchaos)(0xstop),) \
    )) \
    /**/

ADD(11, 13) // 24

#endif

CHAOS_PP_MACHINE_ID

The CHAOS_PP_MACHINE_ID macro indirectly expands to CHAOS_PP_MACHINE.

Usage - C99 Specific 

CHAOS_PP_MACHINE_ID()

Remarks

This macro effectively allows CHAOS_PP_MACHINE to pass through itself without becoming disabled.

CHAOS_PP_MACHINE_

The CHAOS_PP_MACHINE_ macro expands to a λ expression representing CHAOS_PP_MACHINE.

Usage - C99 Specific 

CHAOS_PP_MACHINE_

Remarks

This macro is a shorthand form of the following λ binding: 

CHAOS_PP_LAMBDA(CHAOS_PP_MACHINE_ID)()
