The generics/typeof.h header defines a macro that extracts the type of a generic data structure.

Usage

#include <chaos/preprocessor/generics/typeof.h>

Contents

  • CHAOS_PP_TYPEOF

CHAOS_PP_TYPEOF

The CHAOS_PP_TYPEOF macro extracts the type of its argument.

Usage 

CHAOS_PP_TYPEOF(g)
g

The generic data structure whose type is to extracted.

Remarks

This macro expands to the type of g.

Sample Code

#include <chaos/preprocessor/generics/typeof.h>
#include <chaos/preprocessor/list/core.h>

CHAOS_PP_TYPEOF((CHAOS_PP_LIST) (a, (b, (c, ...)))) // CHAOS_PP_LIST

CHAOS_PP_TYPEOF_ID

The CHAOS_PP_TYPEOF_ID macro indirectly expands to CHAOS_PP_TYPEOF.

Usage 

CHAOS_PP_TYPEOF_ID()

Remarks

This macro exists only for uniformity with other macros that require indirect invocation.

CHAOS_PP_TYPEOF_

The CHAOS_PP_TYPEOF_ macro expands to a λ expression representing CHAOS_PP_TYPEOF.

Usage - C99 Specific 

CHAOS_PP_TYPEOF_

Remarks

This macro is a shorthand form of the following λ binding: 

CHAOS_PP_LAMBDA(CHAOS_PP_TYPEOF)
