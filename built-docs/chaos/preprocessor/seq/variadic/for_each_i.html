The seq/variadic/for_each_i.h header defines an algorithm that repeats a user-specified target for each element in a variadic sequence.

Usage

#include <chaos/preprocessor/seq/variadic/for_each_i.h>

Contents

  • CHAOS_PP_VARIADIC_SEQ_FOR_EACH_I

CHAOS_PP_VARIADIC_SEQ_FOR_EACH_I

The CHAOS_PP_VARIADIC_SEQ_FOR_EACH_I macro repeats the target specified by its first argument for each element in its second argument.

Usage - C99 Specific 

CHAOS_PP_EXPR(CHAOS_PP_VARIADIC_SEQ_FOR_EACH_I(macro, seq))
macro

The target that is to be repeated for each element in seq. This target must be either a λ expression or a macro name that refers to a macro with a signature compatible with the following pseudo-signature: 

macro(s, i, tuple)
[seq]

The variadic sequence for which macro is to be repeated for each element.

Usage - C99 Specific 

CHAOS_PP_EXPR(CHAOS_PP_VARIADIC_SEQ_FOR_EACH_I(macro, seq, ...))
macro

The target that is to be repeated for each element in seq. This target must be either a λ expression or a macro name that refers to a macro with a signature compatible with the following pseudo-signature: 

macro(s, i, tuple, data†)

† The data pseudo-parameter must be either ... or the number of parameters that corresponds to the number of arguments in the auxiliary data, __VA_ARGS__.

[seq]

The variadic sequence for which macro is to be repeated for each element.

[...]

Auxiliary data to be passed through the algorithm to the target.

Remarks

This macro behaves identically to CHAOS_PP_SEQ_FOR_EACH_I except that each element is wrapped in parentheses prior to being passed to the target. For example, instead of the the target being invoked as... 

macro(s, i, elem, state)

it is invoked as... 

macro(s, i, (elem), state)

This macro deduces the current recursion state and invokes CHAOS_PP_VARIADIC_SEQ_FOR_EACH_I_S. If the current recursion state is already known, direct use of CHAOS_PP_VARIADIC_SEQ_FOR_EACH_I_S is more efficient.

This macro may not be used if bypass semantics are in effect.

CHAOS_PP_VARIADIC_SEQ_FOR_EACH_I_ID

The CHAOS_PP_VARIADIC_SEQ_FOR_EACH_I_ID macro indirectly expands to CHAOS_PP_VARIADIC_SEQ_FOR_EACH_I.

Usage - C99 Specific 

CHAOS_PP_VARIADIC_SEQ_FOR_EACH_I_ID()

Remarks

This macro effectively allows CHAOS_PP_VARIADIC_SEQ_FOR_EACH_I to pass through itself without becoming disabled.

CHAOS_PP_VARIADIC_SEQ_FOR_EACH_I_

The CHAOS_PP_VARIADIC_SEQ_FOR_EACH_I_ macro expands to a λ expression representing CHAOS_PP_VARIADIC_SEQ_FOR_EACH_I.

Usage - C99 Specific 

CHAOS_PP_VARIADIC_SEQ_FOR_EACH_I_

Remarks

This macro is a shorthand form of the following λ binding: 

CHAOS_PP_LAMBDA(CHAOS_PP_VARIADIC_SEQ_FOR_EACH_I_ID)()
