The seq/variadic/core.h header defines the core variadic sequence operations necessary to use a variadic sequence as a generic data structure.

Usage

#include <chaos/preprocessor/seq/variadic/core.h>

CHAOS_PP_VARIADIC_SEQ

The CHAOS_PP_VARIADIC_SEQ macro designates the name of the variadic sequence data structure.

Usage - C99 Specific 

CHAOS_PP_VARIADIC_SEQ

Remarks

This macro expands to CHAOS_PP_VARIADIC_SEQ. In other words, the identifier CHAOS_PP_VARIADIC_SEQ is defined as itself.

CHAOS_PP_VARIADIC_SEQ_ID

The CHAOS_PP_VARIADIC_SEQ_ID macro indirectly expands to CHAOS_PP_VARIADIC_SEQ.

Usage - C99 Specific

Remarks

CHAOS_PP_VARIADIC_SEQ_

Usage - C99 Specific

Remarks

This macro is a shorthand form of the following λ binding:

CHAOS_PP_VARIADIC_SEQ_TAG

The tag used to distinguish between data structures.

Remarks

This macro is used by generic algorithms to compare the types of generic data structures. It should be used for no other purpose.

CHAOS_PP_VARIADIC_SEQ_CONS

The CHAOS_PP_VARIADIC_SEQ_CONS macro appends an element to the head of a variadic sequence.

Usage - C99 Specific 

CHAOS_PP_VARIADIC_SEQ_CONS(seq, ...)
[seq]

The variadic sequence to which __VA_ARGS__ is to be appended.

[...]

The element to be appended to the head of seq.

Remarks

This macro expands to a variadic sequence containing the elements of seq with __VA_ARGS__ appended to the head.

See Also

  • CHAOS_PP_ARRAY_CONS
  • CHAOS_PP_CONS
  • CHAOS_PP_LIST_CONS
  • CHAOS_PP_SEQ_CONS
  • CHAOS_PP_STRING_CONS

#include <chaos/preprocessor/config.h>
#include <chaos/preprocessor/seq/variadic/core.h>

CHAOS_PP_VARIADIC_SEQ_CONS((a, b)(c), +, -) // (+, -)(a, b)(c)

CHAOS_PP_VARIADIC_SEQ_ITEM

The CHAOS_PP_VARIADIC_SEQ_ITEM macro is an identity macro used to encapsulate elements.

Usage - C99 Specific 

CHAOS_PP_VARIADIC_SEQ_ITEM(...)
[...]

The element of a variadic sequence to be encapsulated.

Remarks

This macro is used by generic algorithms to encapsulate elements before they are passed to user-defined macros.

This macro expands to its argument wrapped in parentheses (i.e. (__VA_ARGS__)).

CHAOS_PP_VARIADIC_SEQ_IS_CONS

The CHAOS_PP_VARIADIC_SEQ_IS_CONS macro is a predicate that test whether a variadic sequence is non-nil.

Usage - C99 Specific 

CHAOS_PP_VARIADIC_SEQ_IS_CONS(seq)
[seq]

The variadic sequence to be tested.

Remarks

If seq is non-nil, this macro expands to 1. Otherwise, it expands to 0.
