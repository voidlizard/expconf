The seq/drop.h header defines a macro that yields the remainder of a sequence after removing a user-specified number of elements.

Usage

#include <chaos/preprocessor/seq/drop.h>

Contents

  • CHAOS_PP_SEQ_DROP

CHAOS_PP_SEQ_DROP

The CHAOS_PP_SEQ_DROP macro removes the number of elements specified by its first argument from the head of its second argument.

Usage - C99 Specific 

CHAOS_PP_SEQ_DROP(n, seq)
n

The number of elements to be removed. Valid values range from 0 to CHAOS_PP_SEQ_SIZE(seq).

[seq]

The sequence from which elements are to be removed.

Usage - C/C++ Specific 

CHAOS_PP_SEQ_DROP(n, seq)
n

The number of elements to be removed. Valid values range from 0 to CHAOS_PP_SEQ_SIZE(seq).

seq

The sequence from which elements are to be removed.

Remarks

This macro expands to a sequence containing all but the first n elements of seq.

Sample Code

#include <chaos/preprocessor/config.h>
#include <chaos/preprocessor/seq/drop.h>

CHAOS_PP_SEQ_DROP(2, (a)(b)(c)(d)) // (c)(d)
CHAOS_PP_SEQ_DROP(0, (x)(y)(z))    // (x)(y)(z)

#if CHAOS_PP_VARIADICS

CHAOS_PP_SEQ_DROP(0,) //

#endif

CHAOS_PP_SEQ_DROP_ID

The CHAOS_PP_SEQ_DROP_ID macro indirectly expands to CHAOS_PP_SEQ_DROP.

Usage 

CHAOS_PP_SEQ_DROP_ID()

Remarks

This macro effectively allows CHAOS_PP_SEQ_DROP to pass through itself without becoming disabled.

CHAOS_PP_SEQ_DROP_

The CHAOS_PP_SEQ_DROP_ macro expands to a λ expression representing CHAOS_PP_SEQ_DROP.

Usage - C99 Specific 

CHAOS_PP_SEQ_DROP_

Remarks

This macro is a shorthand form of the following λ binding: 

CHAOS_PP_LAMBDA(CHAOS_PP_SEQ_DROP_ID)()
