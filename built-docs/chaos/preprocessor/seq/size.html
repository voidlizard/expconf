The seq/size.h header defines macros that count the elements in a sequence.

Usage

#include <chaos/preprocessor/seq/size.h>

Contents

  • CHAOS_PP_SEQ_SIZE

CHAOS_PP_SEQ_SIZE

The CHAOS_PP_SEQ_SIZE macro counts the elements in its argument.

Usage - C99 Specific 

CHAOS_PP_SEQ_SIZE(seq)
[seq]

The sequence whose elements are to be counted.

Usage - C/C++ Specific 

CHAOS_PP_SEQ_SIZE(seq)
seq

The sequence whose elements are to be counted.

Remarks

This macro expands to the number of elements in seq. If seq contains more than CHAOS_PP_LIMIT_MAG elements, the result is saturated to CHAOS_PP_LIMIT_MAG.

Sample Code

#include <chaos/preprocessor/config.h>
#include <chaos/preprocessor/seq/size.h>

CHAOS_PP_SEQ_SIZE((+)(-)(*)(/)(%)) // 5

#if CHAOS_PP_VARIADICS

CHAOS_PP_SEQ_SIZE((1)(2, 2)(3, 3, 3)) // 3

#endif

CHAOS_PP_SEQ_SIZE_ALT

The CHAOS_PP_SEQ_SIZE_ALT macro counts the elements in its argument.

Usage - C99 Specific 

CHAOS_PP_SEQ_SIZE_ALT(seq)
[seq]

The sequence whose elements are to be counted.

Usage - C/C++ Specific 

CHAOS_PP_SEQ_SIZE_ALT(seq)
seq

The sequence whose elements are to be counted.

Remarks

This macro expands to the number of elements in seq. If seq contains more than CHAOS_PP_LIMIT_MAG elements, the result is saturated to CHAOS_PP_LIMIT_MAG.

The efficiency of the algorithm's use of the recursion backend can be described in terms of the number of recursion steps required to complete the algorithm. The asymptotic upper and lower bounds are Θ, where n is the number of elements in seq.

This macro is identical in function to CHAOS_PP_SEQ_SIZE. However, the internal structure of the algorithm may lead to better performance on longer sequences.

This macro uses bypass semantics. If bypass semantics are already in effect, CHAOS_PP_SEQ_SIZE_ALT_BYPASS must be used instead.

CHAOS_PP_SEQ_SIZE_ALT_BYPASS

The CHAOS_PP_SEQ_SIZE_ALT_BYPASS macro behaves identically to CHAOS_PP_SEQ_SIZE_ALT except that it is parametized by the bypass recursion state.

Usage - C99 Specific 

CHAOS_PP_SEQ_SIZE_ALT_BYPASS(s, seq)
s

The current bypass recursion state.

[seq]

The sequence whose elements are to be counted.

Usage - C/C++ Specific 

CHAOS_PP_SEQ_SIZE_ALT_BYPASS(s, seq)
s

The current bypass recursion state.

seq

The sequence whose elements are to be counted.

Remarks

If bypass semantics are already in effect, this macro must be used instead of CHAOS_PP_SEQ_SIZE_ALT.
