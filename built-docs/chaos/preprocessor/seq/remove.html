The seq/remove.h header defines a macro that removes an element from a sequence.

Usage

#include <chaos/preprocessor/seq/remove.h>

Contents

  • CHAOS_PP_SEQ_REMOVE

CHAOS_PP_SEQ_REMOVE

The CHAOS_PP_SEQ_REMOVE macro removes the element at the index specified by its first argument from its second argument.

Usage 

CHAOS_PP_SEQ_REMOVE(i, seq)
i

The zero-based index of the element to be removed. Valid values range from 0 to CHAOS_PP_SEQ_SIZE(seq) - 1.

seq

The sequence from which an element is to be removed.

Remarks

This macro expands to a sequence containing all of the elements of seq except the i-th element.

Sample Code

#include <chaos/preprocessor/config.h>
#include <chaos/preprocessor/seq/remove.h>

CHAOS_PP_SEQ_REMOVE(2, (a)(b)(c)(d)(e)) // (a)(b)(d)(e)

#if CHAOS_PP_VARIADICS

CHAOS_PP_SEQ_REMOVE(1, (a, b)(c)(d, e, f)) // (a, b)(d, e, f)

#endif

CHAOS_PP_SEQ_REMOVE_ID

The CHAOS_PP_SEQ_REMOVE_ID macro indirectly expands to CHAOS_PP_SEQ_REMOVE.

Usage 

CHAOS_PP_SEQ_REMOVE_ID()

Remarks

This macro effectively allows CHAOS_PP_SEQ_REMOVE to pass through itself without becoming disabled.

CHAOS_PP_SEQ_REMOVE_

The CHAOS_PP_SEQ_REMOVE_ macro expands to a λ expression representing CHAOS_PP_SEQ_REMOVE.

Usage - C99 Specific 

CHAOS_PP_SEQ_REMOVE_

Remarks

This macro is a shorthand form of the following λ binding: 

CHAOS_PP_LAMBDA(CHAOS_PP_SEQ_REMOVE_ID)()
