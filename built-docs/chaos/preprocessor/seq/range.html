The seq/range.h header defines a macro that extracts a range of elements from a sequence.

Usage

#include <chaos/preprocessor/seq/range.h>

Contents

  • CHAOS_PP_SEQ_RANGE

CHAOS_PP_SEQ_RANGE

The CHAOS_PP_SEQ_RANGE macro extracts the range of elements specified by its first two arguments from its third argument.

Usage - C99 Specific 

CHAOS_PP_SEQ_RANGE(i, len, seq)
i

The zero-based index of the first element to be extracted. Valid values range from 0 to CHAOS_PP_SEQ_SIZE(seq) - 1.

len

The number of elements to be extracted. Valid values range from 0 to CHAOS_PP_SEQ_SIZE(seq) - i.

[seq]

The sequence from which elements are to be extracted.

Usage - C/C++ Specific 

CHAOS_PP_SEQ_RANGE(i, len, seq)
i

The zero-based index of the first element to be extracted. Valid values range from 0 to CHAOS_PP_SEQ_SIZE(seq) - 1.

len

The number of elements to be extracted. Valid values range from 0 to CHAOS_PP_SEQ_SIZE(seq) - i.

seq

The sequence from which elements are to be extracted.

Remarks

This macro expands to a sequence containing len elements from seq starting at the i-th element of seq. Consequently, seq must contain the entire range specified by i and len.

Sample Code

#include <chaos/preprocessor/config.h>
#include <chaos/preprocessor/seq/range.h>

CHAOS_PP_SEQ_RANGE(1, 3, (a)(b)(c)(d)(e)) // (b)(c)(d)

#if CHAOS_PP_VARIADICS

CHAOS_PP_SEQ_RANGE(0, 0,) //
CHAOS_PP_SEQ_RANGE(1, 2, (a, b)(c)(d, e)(f)) // (c)(d, e)

#endif

CHAOS_PP_SEQ_RANGE_ID

The CHAOS_PP_SEQ_RANGE_ID macro indirectly expands to CHAOS_PP_SEQ_RANGE.

Usage 

CHAOS_PP_SEQ_RANGE_ID()

Remarks

This macro effectively allows CHAOS_PP_SEQ_RANGE to pass through itself without becoming disabled.

CHAOS_PP_SEQ_RANGE_

The CHAOS_PP_SEQ_RANGE_ macro expands to a λ expression representing CHAOS_PP_SEQ_RANGE.

Usage - C99 Specific 

CHAOS_PP_SEQ_RANGE_

Remarks

This macro is a shorthand form of the following λ binding: 

CHAOS_PP_LAMBDA(CHAOS_PP_SEQ_RANGE_ID)()
