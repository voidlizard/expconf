CHAOS_PP_VALUE

The CHAOS_PP_VALUE macro is a user-defined named external argument that specifies the expression to be evaluated and assigned to a slot index.

Usage 

#define CHAOS_PP_VALUE expr
expr

The expression that is to be evaluated.

Remarks

This macro is defined by the user immediately preceding an invocation of CHAOS_PP_ASSIGN_SLOT, which, in turn, evaluates the expression contained in CHAOS_PP_VALUE and assigns it to a slot index. After that assignment, CHAOS_PP_VALUE is undefined automatically.

See Also

  • CHAOS_PP_SLOT

CHAOS_PP_ASSIGN_SLOT

The CHAOS_PP_ASSIGN_SLOT macro fully evaluates a numeric expression.

Usage 

??=include CHAOS_PP_ASSIGN_SLOT(i)
i

The slot index that is to be assigned. Valid values range from 1 to CHAOS_PP_LIMIT_SLOT.

Remarks

This macro evaluates the numeric expression contained in the named external argument CHAOS_PP_VALUE and assigns it to the i-th evaluated slot. If CHAOS_PP_VALUE is not defined prior to the use of this macro, it is given the default value of 0. The numeric expression must evaluate to a value that is 1 to CHAOS_PP_LIMIT_SLOT_VALUE digits (base-10). The range of the expression is also possibly constricted by the preprocessor in use. Specifically, the preprocessor must be capable of evaluating the expression on an #if or #elif directive.

After CHAOS_PP_ASSIGN_SLOT has assigned the resulting value of CHAOS_PP_VALUE to the i-th evaluated slot, it undefines CHAOS_PP_VALUE.

See Also

  • CHAOS_PP_SLOT
  • CHAOS_PP_VALUE

Sample Code

#include <chaos/preprocessor/slot/slot.h>

#define CHAOS_PP_VALUE 1 + 2 + 3
??=include CHAOS_PP_ASSIGN_SLOT(1)

CHAOS_PP_SLOT(1) // 6

CHAOS_PP_ASSIGN_SLOT_ID

The CHAOS_PP_ASSIGN_SLOT_ID macro indirectly expands to CHAOS_PP_ASSIGN_SLOT.

Usage 

CHAOS_PP_ASSIGN_SLOT_ID()

Remarks

This macro exists only for uniformity with other macros that require indirect invocation.

CHAOS_PP_ASSIGN_SLOT_

The CHAOS_PP_ASSIGN_SLOT_ macro expands to a λ expression representing CHAOS_PP_ASSIGN_SLOT.

Usage - C99 Specific 

CHAOS_PP_ASSIGN_SLOT_

Remarks

This macro is a shorthand form of the following λ binding: 

CHAOS_PP_LAMBDA(CHAOS_PP_ASSIGN_SLOT)

CHAOS_PP_SLOT

The CHAOS_PP_SLOT macro retrieves a value previously evaluated by CHAOS_PP_ASSIGN_SLOT.

Usage 

CHAOS_PP_SLOT(i)
i

The slot index of the value that is to be retrieved. Valid values range from 1 to CHAOS_PP_LIMIT_SLOT.

Remarks

This macro retrieves the value that was previously assigned to the slot index i. If the slot index specified by i has never been assigned, this macro expands to 0.

See Also

  • CHAOS_PP_ASSIGN_SLOT
  • CHAOS_PP_VALUE

Sample Code

#include <chaos/preprocessor/slot/slot.h>

#define X() 4

#define CHAOS_PP_VALUE 1 + 2 + 3 + X()
??=include CHAOS_PP_ASSIGN_SLOT(1)

#undef X

CHAOS_PP_SLOT(1) // 10

#define CHAOS_PP_VALUE CHAOS_PP_SLOT(1) * CHAOS_PP_SLOT(1)
??=include CHAOS_PP_ASSIGN_SLOT(1)

CHAOS_PP_SLOT(1) // 100
