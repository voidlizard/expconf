The slot/counter.h header defines macros used generate numbers that are unique within a translation unit.

Usage

#include <chaos/preprocessor/slot/counter.h>

Contents

  • CHAOS_PP_COUNTER

CHAOS_PP_COUNTER

The CHAOS_PP_COUNTER macro expands to the current counter value.

Usage 

CHAOS_PP_COUNTER

Remarks

This macro expands to an integer literal. Its initial value defaults to 0. This initial value (per translation unit) can be overridden by defining CHAOS_PP_SEED. Each usage of CHAOS_PP_UPDATE_COUNTER increases the current value by one. Between usages of CHAOS_PP_UPDATE_COUNTER, the value of CHAOS_PP_COUNTER is constant.

See Also

  • CHAOS_PP_SEED
  • CHAOS_PP_UPDATE_COUNTER

Sample Code

#include <chaos/preprocessor/cat.h>
#include <chaos/preprocessor/slot/counter.h>

??=include CHAOS_PP_UPDATE_COUNTER()

struct CHAOS_PP_CAT(register_, CHAOS_PP_COUNTER) {
    inline CHAOS_PP_CAT(register_, CHAOS_PP_COUNTER)(void) {
        return;
    }
    // ...
};

??=include CHAOS_PP_UPDATE_COUNTER()

struct CHAOS_PP_CAT(register_, CHAOS_PP_COUNTER) {
    inline CHAOS_PP_CAT(register_, CHAOS_PP_COUNTER)(void) {
        return;
    }
    // ...
};

CHAOS_PP_COUNTER_ID

The CHAOS_PP_COUNTER_ID macro indirectly expands to CHAOS_PP_COUNTER.

Usage 

CHAOS_PP_COUNTER_ID()

Remarks

This macro effectively allows CHAOS_PP_COUNTER to be passed around without expansion.

CHAOS_PP_COUNTER_

The CHAOS_PP_COUNTER_ macro expands to a λ expression representing CHAOS_PP_COUNTER.

Usage - C99 Specific 

CHAOS_PP_COUNTER_

Remarks

This macro is a shorthand form of the following λ binding: 

CHAOS_PP_LAMBDA(CHAOS_PP_COUNTER_ID)()

CHAOS_PP_SEED

The CHAOS_PP_SEED macro defines the initial value (in a translation unit) of CHAOS_PP_COUNTER.

Usage 

#define CHAOS_PP_SEED x
x

The integer literal to be used as the initial value of CHAOS_PP_COUNTER.

Remarks

If defined by the user, this macro must be defined prior to including any library headers. If it is not defined by the user, it defaults to 0.

This macro can be defined on the command line such that unique numbers can be generated across multiple translation units.
