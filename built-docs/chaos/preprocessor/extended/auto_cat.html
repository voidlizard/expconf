The extended/auto_cat.h header defines a recursive concatenation macro.

Usage

#include <chaos/preprocessor/extended/auto_cat.h>

Contents

  • CHAOS_PP_AUTO_CAT

CHAOS_PP_AUTO_CAT

The CHAOS_PP_AUTO_CAT macro is a concatenation macro that employs automatic recursion.

Usage - C99 Specific 

CHAOS_PP_AUTO_CAT(a, ...)
[a]

The left operand of the concatenation.

[...]

The right operand of the concatenation.

Usage - C/C++ Specific 

CHAOS_PP_AUTO_CAT(a, b)
a

The left operand of the concatenation.

b

The right operand of the concatenation.

Remarks

This macro behaves identically to CHAOS_PP_CAT except that it utilizes automatic recursion. It supports up to CHAOS_PP_LIMIT_AUTO_CAT nested concatenations.

Sample Code

#include <chaos/preprocessor/extended/auto_cat.h>

#define AA CHAOS_PP_AUTO_CAT(B, B)
#define BB CHAOS_PP_AUTO_CAT(C, C)
#define CC 1

CHAOS_PP_AUTO_CAT(A, A) // 1

CHAOS_PP_AUTO_CAT_ID

The CHAOS_PP_AUTO_CAT_ID macro indirectly expands to CHAOS_PP_AUTO_CAT.

Usage 

CHAOS_PP_AUTO_CAT_ID()

Remarks

This macro effectively allows CHAOS_PP_AUTO_CAT to be passed around without expansion.

CHAOS_PP_AUTO_CAT_

The CHAOS_PP_AUTO_CAT_ macro expands to a λ expression representing CHAOS_PP_AUTO_CAT.

Usage - C99 Specific 

CHAOS_PP_AUTO_CAT_

Remarks

This macro is a shorthand form of the following λ binding: 

CHAOS_PP_LAMBDA(CHAOS_PP_AUTO_CAT_ID)()
