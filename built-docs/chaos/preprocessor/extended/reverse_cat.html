The extended/reverse_cat.h header defines concatenation macros that reverse their operands prior to concatenation.

Usage

#include <chaos/preprocessor/extended/reverse_cat.h>

Contents

  • CHAOS_PP_PRIMITIVE_REVERSE_CAT
  • CHAOS_PP_REVERSE_CAT

CHAOS_PP_REVERSE_CAT

The CHAOS_PP_REVERSE_CAT macro concatenates its arguments after they have been expanded.

Usage - C99 Specific 

CHAOS_PP_REVERSE_CAT(a, ...)
[a]

The right operand of the concatenation.

[...]

The left operand of the concatenation.

Usage - C/C++ Specific 

CHAOS_PP_REVERSE_CAT(a, b)
a

The right operand of the concatenation.

b

The left operand of the concatenation.

Remarks

This macro behaves identically to CHAOS_PP_CAT except that it concatenates the rightmost preprocessing token of a to the leftmost preprocessing token of b or __VA_ARGS__. 

__VA_ARGS__ ## a // C99
b ## a           // C and C++

Sample Code

#include <chaos/preprocessor/extended/reverse_cat.h>

#define A 1
#define B 2

CHAOS_PP_REVERSE_CAT(A, B) // 21

CHAOS_PP_REVERSE_CAT_ID

The CHAOS_PP_REVERSE_CAT_ID macro indirectly expands to CHAOS_PP_REVERSE_CAT.

Usage 

CHAOS_PP_REVERSE_CAT_ID()

Remarks

This macro effectively allows CHAOS_PP_REVERSE_CAT to pass through itself without becoming disabled.

CHAOS_PP_REVERSE_CAT_

The CHAOS_PP_REVERSE_CAT_ macro expands to a λ expression representing CHAOS_PP_REVERSE_CAT.

Usage - C99 Specific 

CHAOS_PP_REVERSE_CAT_

Remarks

This macro is a shorthand form of the following λ binding: 

CHAOS_PP_LAMBDA(CHAOS_PP_REVERSE_CAT_ID)()
