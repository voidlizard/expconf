CHAOS_PP_ARBITRARY_SIGN

The CHAOS_PP_ARBITRARY_SIGN macro extracts the sign of an arbitrary-precision value.

Usage 

CHAOS_PP_ARBITRARY_SIGN(x)
x

The arbitrary-precision value whose sign is to be extracted.

Remarks

If x is an arbitrary-precision value representing a negative number, this macro expands to 1. Otherwise, it expands to 0.

Sample Code

#include <chaos/preprocessor/arbitrary/literal.h>
#include <chaos/preprocessor/arbitrary/sign.h>

CHAOS_PP_ARBITRARY_SIGN(CHAOS_PP_ARBITRARY_LITERAL(1 2 3)) // 0
CHAOS_PP_ARBITRARY_SIGN(CHAOS_PP_ARBITRARY_LITERAL((-)1))  // 1

CHAOS_PP_ARBITRARY_SIGN_ID

The CHAOS_PP_ARBITRARY_SIGN_ID macro indirectly expands to CHAOS_PP_ARBITRARY_SIGN.

Usage 

CHAOS_PP_ARBITRARY_SIGN_ID()

Remarks

This macro exists only for uniformity with other macros that require indirect invocation.

CHAOS_PP_ARBITRARY_SIGN_

The CHAOS_PP_ARBITRARY_SIGN_ macro expands to a λ expression representing CHAOS_PP_ARBITRARY_SIGN.

Usage - C99 Specific 

CHAOS_PP_ARBITRARY_SIGN_

Remarks

This macro is a shorthand form of the following λ binding: 

CHAOS_PP_LAMBDA(CHAOS_PP_ARBITRARY_SIGN)
