CHAOS_PP_ARBITRARY_MOD

The CHAOS_PP_ARBITRARY_MOD macro computes the remainder produced by dividing two arbitrary-precision values.

Usage 

CHAOS_PP_ARBITRARY_MOD(x, y)
x

The base.

y

The modulus.

Remarks

This macro expands to an arbitrary-precision value that represents the remainder produced by dividing x by y. If y is an arbitrary-precision value representing zero, this macro intentionally produces an error.

Sample Code

#include <chaos/preprocessor/arbitrary/demote.h>
#include <chaos/preprocessor/arbitrary/mod.h>

CHAOS_PP_ARBITRARY_DEMOTE(CHAOS_PP_ARBITRARY_MOD(
    (1)(2)(3)(4),
    (5)(0)
))
// 34

CHAOS_PP_ARBITRARY_MOD_ID

The CHAOS_PP_ARBITRARY_MOD_ID macro indirectly expands to CHAOS_PP_ARBITRARY_MOD.

Usage 

CHAOS_PP_ARBITRARY_MOD_ID()

Remarks

This macro exists only for uniformity with other macros that require indirect invocation.

CHAOS_PP_ARBITRARY_MOD_

The CHAOS_PP_ARBITRARY_MOD_ macro expands to a λ expression representing CHAOS_PP_ARBITRARY_MOD.

Usage - C99 Specific 

CHAOS_PP_ARBITRARY_MOD_

Remarks

This macro is a shorthand form of the following λ binding: 

CHAOS_PP_LAMBDA(CHAOS_PP_ARBITRARY_MOD)
