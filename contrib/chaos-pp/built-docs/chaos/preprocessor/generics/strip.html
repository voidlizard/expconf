The generics/strip.h header defines a macro that removes the type prefix of a generic data structure.

Usage

#include <chaos/preprocessor/generics/strip.h>

Contents

  • CHAOS_PP_STRIP

CHAOS_PP_STRIP

The CHAOS_PP_STRIP macro removes the type prefix from its argument.

Usage 

CHAOS_PP_STRIP(g)
g

The generic data structure whose type prefix is to be stripped.

Remarks

This macro expands to the underlying data structure of g.

Sample Code

#include <chaos/preprocessor/config.h>
#include <chaos/preprocessor/generics/core.h>
#include <chaos/preprocessor/generics/strip.h>
#include <chaos/preprocessor/tuple/core.h>

#if CHAOS_PP_VARIADICS

#define X \
    CHAOS_PP_CONS( \
        CHAOS_PP_CONS( \
            CHAOS_PP_NIL(CHAOS_PP_TUPLE), b \
        ), a \
    ) \
    /**/

X                 // (CHAOS_PP_TUPLE) (a, b)
CHAOS_PP_STRIP(X) // (a, b)

#endif

CHAOS_PP_STRIP_ID

The CHAOS_PP_STRIP_ID macro indirectly expands to CHAOS_PP_STRIP.

Usage 

CHAOS_PP_STRIP_ID()

Remarks

This macro effectively allows CHAOS_PP_STRIP to pass through itself without becoming disabled.

CHAOS_PP_STRIP_

The CHAOS_PP_STRIP_ macro expands to a λ expression representing CHAOS_PP_STRIP.

Usage - C99 Specific 

CHAOS_PP_STRIP_

Remarks

This macro is a shorthand form of the following λ binding: 

CHAOS_PP_LAMBDA(CHAOS_PP_STRIP_ID)()
