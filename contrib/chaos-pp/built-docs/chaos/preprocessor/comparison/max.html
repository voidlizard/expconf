The comparison/max.h header defines a macro that expands to the maximum of two values.

Usage

#include <chaos/preprocessor/comparison/max.h>

Contents

  • CHAOS_PP_MAX

CHAOS_PP_MAX

The CHAOS_PP_MAX macro expands to the maximum of its two arguments.

Usage 

CHAOS_PP_MAX(x, y)
x

The first operand. Valid values range from 0 to CHAOS_PP_LIMIT_MAG.

y

The second operand. Valid values range from 0 to CHAOS_PP_LIMIT_MAG.

Remarks

If x is greater than or equal to y, this macro expands to x. Otherwise, it expands to y.

See Also

  • CHAOS_PP_MIN

Sample Code

#include <chaos/preprocessor/comparison/max.h>

CHAOS_PP_MAX(2, 4) // 4

CHAOS_PP_MAX_ID

The CHAOS_PP_MAX_ID macro indirectly expands to CHAOS_PP_MAX.

Usage 

CHAOS_PP_MAX_ID()

Remarks

This macro exists only for uniformity with other macros that require indirect invocation.

CHAOS_PP_MAX_

The CHAOS_PP_MAX_ macro expands to a λ expression representing CHAOS_PP_MAX.

Usage - C99 Specific 

CHAOS_PP_MAX_

Remarks

This macro is a shorthand form of the following λ binding: 

CHAOS_PP_LAMBDA(CHAOS_PP_MAX)
