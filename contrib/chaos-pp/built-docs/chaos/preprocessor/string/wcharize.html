The string/wcharize.h header defines a macro that converts a character to a wide character literal.

Usage

#include <chaos/preprocessor/string/wcharize.h>

Contents

  • CHAOS_PP_WCHARIZE

CHAOS_PP_WCHARIZE

The CHAOS_PP_WCHARIZE macro converts its argument to a wide character literal.

Usage 

CHAOS_PP_WCHARIZE(c)
c

The character to be converted. Valid values are the underscore (_), the decimal numerals (0 through 9), and the lowercase and uppercase letters (a through z and A through Z).

Remarks

This macro is not a general-purpose "charizing" macro. Input to this macro is restricted to characters that can be elements in a string data structure. For example, 

CHAOS_PP_WCHARIZE(+) // invalid

Sample Code

#include <chaos/preprocessor/algorithm/for_each.h>
#include <chaos/preprocessor/recursion/expr.h>
#include <chaos/preprocessor/string/core.h>
#include <chaos/preprocessor/string/wcharize.h>

#define TO_WCHAR_SEQ(string) \
    CHAOS_PP_EXPR(CHAOS_PP_FOR_EACH( \
        TO_CHAR_SEQ_I, (CHAOS_PP_STRING) string, ~ \
    )) \
    /**/

#define TO_WCHAR_SEQ_I(s, c, _) (CHAOS_PP_WCHARIZE(c))

TO_CHAR_SEQ(t e m p l a t e) // (L't')(L'e')(L'm')(L'p')(L'l')(L'a')(L't')(L'e')

CHAOS_PP_WCHARIZE_ID

The CHAOS_PP_WCHARIZE_ID macro indirectly expands to CHAOS_PP_WCHARIZE.

Usage 

CHAOS_PP_WCHARIZE_ID()

Remarks

This macro exists only for uniformity with other macros that require indirect invocation.

CHAOS_PP_WCHARIZE_

The CHAOS_PP_WCHARIZE_ macro expands to a λ expression representing CHAOS_PP_WCHARIZE.

Usage - C99 Specific 

CHAOS_PP_WCHARIZE_

Remarks

This macro is a shorthand form of the following λ binding: 

CHAOS_PP_LAMBDA(CHAOS_PP_WCHARIZE)
