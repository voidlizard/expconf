The string/ucharize.h header defines a macro that converts a character to a UTF-16 or UTF-32 character literal.

Usage

#include <chaos/preprocessor/string/ucharize.h>

Contents

  • CHAOS_PP_UCHARIZE

CHAOS_PP_UCHARIZE

The CHAOS_PP_UCHARIZE macro converts its argument to a UTF-16 or UTF-32 character literal.

Usage - C++0x Specific 

CHAOS_PP_UCHARIZE(n)(c)
n

The type of encoding of the resulting character literal. Valid values are 16 and 32.

c

The character to be converted. Valid values are the underscore (_), the decimal numerals (0 through 9), and the lowercase and uppercase letters (a through z and A through Z).

Remarks

This macro is not a general-purpose "charizing" macro. Input to this macro is restricted to characters that can be elements in a string data structure. For example, 

CHAOS_PP_UCHARIZE(16)(+) // invalid

Sample Code

#include <chaos/preprocessor/string/ucharize.h>

CHAOS_PP_UCHARIZE(32)(a) // U'a'

CHAOS_PP_UCHARIZE_ID

The CHAOS_PP_UCHARIZE_ID macro indirectly expands to CHAOS_PP_UCHARIZE.

Usage 

CHAOS_PP_UCHARIZE_ID()

Remarks

This macro exists only for uniformity with other macros that require indirect invocation.

CHAOS_PP_UCHARIZE_

The CHAOS_PP_UCHARIZE_ macro expands to a λ expression representing CHAOS_PP_UCHARIZE.

Usage - C++0x Specific 

CHAOS_PP_UCHARIZE_

Remarks

This macro is a shorthand form of the following λ binding: 

CHAOS_PP_LAMBDA(CHAOS_PP_UCHARIZE)
