The seq/auto/filter.h header defines an algorithm that filters elements from a sequence according to a user-specified criteria.

Usage

#include <chaos/preprocessor/seq/auto/filter.h>

Contents

  • CHAOS_PP_SEQ_AUTO_FILTER

CHAOS_PP_SEQ_AUTO_FILTER

The CHAOS_PP_SEQ_AUTO_FILTER macro filters elements from its second argument according to the criteria specified by its first argument.

Usage - C99 Specific 

CHAOS_PP_SEQ_AUTO_FILTER(pred, seq)
pred

The predicate that determines whether an element is to remain in the resulting sequence. This predicate must be either a λ expression or a macro name that refers to a macro with a signature compatible with the following pseudo-signature: 

pred(s, elem†)

† If seq contains elements that have varying arities, elem must be .... Otherwise, elem must be the number of parameters that corresponds to the arity of each element in seq.

[seq]

The sequence to be filtered.

Usage - C99 Specific 

CHAOS_PP_SEQ_AUTO_FILTER(pred, seq, ...)
pred

The predicate that determines whether an element is to remain in the resulting sequence. This predicate must be either a λ expression or a macro name that refers to a macro with a signature compatible with the following pseudo-signature: 

pred(s, elem†, data‡)

† The elem pseudo-parameter must be the number of parameters that corresponds to the arity of each element in seq.

‡ The data pseudo-parameter must be either ... or the number of parameters that corresponds to the number of arguments in the auxiliary data, __VA_ARGS__.

[seq]

The sequence to be filtered.

[...]

Auxiliary data to be passed through the algorithm to the predicate.

Remarks

The CHAOS_PP_SEQ_AUTO_FILTER macro is a higher-order thunk that deduces the current higher-order entry point and invokes CHAOS_PP_SEQ_FILTER.

This macro may not be used if bypass semantics are in effect.

Sample Code

#include <chaos/preprocessor/config.h>
#include <chaos/preprocessor/lambda/ops.h>
#include <chaos/preprocessor/logical/bool.h>
#include <chaos/preprocessor/seq/auto/filter.h>

#if CHAOS_PP_VARIADICS

CHAOS_PP_SEQ_AUTO_FILTER(
    CHAOS_PP_BOOL_(CHAOS_PP_ARG(1)), (1)(0)(2)(0)(3)(0)(4)(0)(5)
)
// (1)(2)(3)(4)(5)

#endif

CHAOS_PP_SEQ_AUTO_FILTER_ID

The CHAOS_PP_SEQ_AUTO_FILTER_ID macro indirectly expands to CHAOS_PP_SEQ_AUTO_FILTER.

Usage - C99 Specific 

CHAOS_PP_SEQ_AUTO_FILTER_ID()

Remarks

This macro effectively allows CHAOS_PP_SEQ_AUTO_FILTER to be passed around without expansion.

CHAOS_PP_SEQ_AUTO_FILTER_

The CHAOS_PP_SEQ_AUTO_FILTER_ macro expands to a λ expression representing CHAOS_PP_SEQ_AUTO_FILTER.

Usage - C99 Specific 

CHAOS_PP_SEQ_AUTO_FILTER_

Remarks

This macro is a shorthand form of the following λ binding: 

CHAOS_PP_LAMBDA(CHAOS_PP_SEQ_AUTO_FILTER_ID)()
