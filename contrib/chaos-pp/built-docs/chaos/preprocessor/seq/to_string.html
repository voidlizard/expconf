The seq/to_string.h header defines a macro that converts a sequence to a string.

Usage

#include <chaos/preprocessor/seq/to_string.h>

Contents

  • CHAOS_PP_SEQ_TO_STRING

CHAOS_PP_SEQ_TO_STRING

The CHAOS_PP_SEQ_TO_STRING macro converts a sequence to a string.

Usage - C99 Specific 

CHAOS_PP_SEQ_TO_STRING(seq)
[seq]

The sequence to be converted.

Usage - C/C++ Specific 

CHAOS_PP_SEQ_TO_STRING(seq)
seq

The sequence to be converted.

Remarks

This macro expands to a string containing the elements of seq.

A string may only contain letters, digits, and underscores. This macro makes no attempt to verify that the resulting string is actually a valid string. 

CHAOS_PP_SEQ_TO_STRING((+)(-)) // invalid

In C99, sequences can contain elements of arities other than one. If such a sequence is converted to a string, the resulting string will be invalid. 

CHAOS_PP_SEQ_TO_STRING((a, b)(c)) // invalid

This macro makes no attempt to diagnose such a situation.

Sample Code

#include <chaos/preprocessor/seq/to_string.h>

CHAOS_PP_SEQ_TO_STRING((a)(b)(c)(d)) // a b c d

CHAOS_PP_SEQ_TO_STRING_ID

The CHAOS_PP_SEQ_TO_STRING_ID macro indirectly expands to CHAOS_PP_SEQ_TO_STRING.

Usage 

CHAOS_PP_SEQ_TO_STRING_ID()

Remarks

This macro effectively allows CHAOS_PP_SEQ_TO_STRING to pass through itself without becoming disabled.

CHAOS_PP_SEQ_TO_STRING_

The CHAOS_PP_SEQ_TO_STRING_ macro expands to a λ expression representing CHAOS_PP_SEQ_TO_STRING.

Usage - C99 Specific 

CHAOS_PP_SEQ_TO_STRING_

Remarks

This macro is a shorthand form of the following λ binding: 

CHAOS_PP_LAMBDA(CHAOS_PP_SEQ_TO_STRING_ID)()
