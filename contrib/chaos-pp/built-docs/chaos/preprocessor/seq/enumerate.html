The seq/enumerate.h header defines a macro that enumerates the elements of a sequence.

Usage

#include <chaos/preprocessor/seq/enumerate.h>

Contents

  • CHAOS_PP_SEQ_ENUMERATE

CHAOS_PP_SEQ_ENUMERATE

The CHAOS_PP_SEQ_ENUMERATE macro enumerates the elements of its argument.

Usage - C99 Specific 

CHAOS_PP_SEQ_ENUMERATE(seq)
[seq]

The sequence whose elements are to be enumerated.

Usage - C/C++ Specific 

CHAOS_PP_SEQ_ENUMERATE(seq)
seq

The sequence whose elements are to be enumerated.

Remarks

This macro expands to a comma-separated list containing the elements of seq.

This macro uses sequential iteration. As a result, the algorithm is theoretically capable of processing sequences containing an unrestricted number of elements.

Sample Code

#include <chaos/preprocessor/config.h>
#include <chaos/preprocessor/seq/enumerate.h>

CHAOS_PP_SEQ_ENUMERATE((a)(b)(c)) // a, b, c

#if CHAOS_PP_VARIADICS

CHAOS_PP_SEQ_ENUMERATE() //
CHAOS_PP_SEQ_ENUMERATE((a, b)(c)) // a, b, c

#endif

CHAOS_PP_SEQ_ENUMERATE_ID

The CHAOS_PP_SEQ_ENUMERATE_ID macro indirectly expands to CHAOS_PP_SEQ_ENUMERATE.

Usage 

CHAOS_PP_SEQ_ENUMERATE_ID()

Remarks

This macro effectively allows CHAOS_PP_SEQ_ENUMERATE to pass through itself without becoming disabled.

CHAOS_PP_SEQ_ENUMERATE_

The CHAOS_PP_SEQ_ENUMERATE_ macro expands to a λ expression representing CHAOS_PP_SEQ_ENUMERATE.

Usage - C99 Specific 

CHAOS_PP_SEQ_ENUMERATE_

Remarks

This macro is a shorthand form of the following λ binding: 

CHAOS_PP_LAMBDA(CHAOS_PP_SEQ_ENUMERATE_ID)()
