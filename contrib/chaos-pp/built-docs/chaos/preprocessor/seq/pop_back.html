The seq/pop_back.h header defines a macro that removes the last element from a sequence.

Usage

#include <chaos/preprocessor/seq/pop_back.h>

Contents

  • CHAOS_PP_SEQ_POP_BACK

CHAOS_PP_SEQ_POP_BACK

The CHAOS_PP_SEQ_POP_BACK macro removes the last element from its argument.

Usage 

CHAOS_PP_SEQ_POP_BACK(seq)
seq

The non-nil sequence whose last element is to be removed.

Remarks

This macro expands to a sequence that contains all of the elements of seq except the last.

This macro uses sequential iteration. As a result, the algorithm is theoretically capable of processing sequences containing an unrestricted number of elements.

Sample Code

#include <chaos/preprocessor/config.h>
#include <chaos/preprocessor/seq/pop_back.h>

CHAOS_PP_SEQ_POP_BACK((a)(b)(c)) // (a)(b)

#if CHAOS_PP_VARIADICS

CHAOS_PP_SEQ_POP_BACK((a, b)(c)(d, e, f)) // (a, b)(c)

#endif

CHAOS_PP_SEQ_POP_BACK_ID

The CHAOS_PP_SEQ_POP_BACK_ID macro indirectly expands to CHAOS_PP_SEQ_POP_BACK.

Usage 

CHAOS_PP_SEQ_POP_BACK_ID()

Remarks

This macro effectively allows CHAOS_PP_SEQ_POP_BACK to pass through itself without becoming disabled.

CHAOS_PP_SEQ_POP_BACK_

The CHAOS_PP_SEQ_POP_BACK_ macro expands to a λ expression representing CHAOS_PP_SEQ_POP_BACK.

Usage - C99 Specific 

CHAOS_PP_SEQ_POP_BACK_

Remarks

This macro is a shorthand form of the following λ binding: 

CHAOS_PP_LAMBDA(CHAOS_PP_SEQ_POP_BACK_ID)()
