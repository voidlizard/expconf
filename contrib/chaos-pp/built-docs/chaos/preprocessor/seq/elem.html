Usage

#include <chaos/preprocessor/seq/elem.h>

Contents

CHAOS_PP_SEQ_ELEM

The CHAOS_PP_SEQ_ELEM macro extracts an element from a sequence.

Usage 

CHAOS_PP_SEQ_ELEM(i, seq)
i

The zero-based index of the element to be extracted. Valid values range from 0 to CHAOS_PP_SEQ_SIZE(seq) - 1.

seq

The sequence from which an element is to be extracted.

Remarks

This macro extracts the i-th element of seq.

Sample Code

#include <chaos/preprocessor/config.h>
#include <chaos/preprocessor/seq/elem.h>

#define A (a)(b)(c)

CHAOS_PP_SEQ_ELEM(0, A) // a
CHAOS_PP_SEQ_ELEM(1, A) // b
CHAOS_PP_SEQ_ELEM(2, A) // c

#if CHAOS_PP_VARIADICS

#define B (a, b)(c)(d, e, f)

CHAOS_PP_SEQ_ELEM(0, B) // a, b
CHAOS_PP_SEQ_ELEM(1, B) // c
CHAOS_PP_SEQ_ELEM(2, B) // d, e, f

#endif

CHAOS_PP_SEQ_ELEM_ID

The CHAOS_PP_SEQ_ELEM_ID macro indirectly expands to CHAOS_PP_SEQ_ELEM.

Usage 

CHAOS_PP_SEQ_ELEM_ID()

Remarks

This macro effectively allows CHAOS_PP_SEQ_ELEM to pass through itself without becoming disabled.

CHAOS_PP_SEQ_ELEM_

The CHAOS_PP_SEQ_ELEM_ macro expands to a λ expression representing CHAOS_PP_SEQ_ELEM.

Usage - C99 Specific 

CHAOS_PP_SEQ_ELEM_

Remarks

This macro is a shorthand form of the following λ binding: 

CHAOS_PP_LAMBDA(CHAOS_PP_SEQ_ELEM_ID)()
