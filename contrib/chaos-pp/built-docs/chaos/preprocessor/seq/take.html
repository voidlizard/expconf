The seq/take.h header defines a macro that extracts a user-specified number of elements from the beginning of a sequence.

Usage

#include <chaos/preprocessor/seq/take.h>

Contents

  • CHAOS_PP_SEQ_TAKE

CHAOS_PP_SEQ_TAKE

The CHAOS_PP_SEQ_TAKE macro extracts the number of elements specified by its first argument from it second argument.

Usage - C99 Specific 

CHAOS_PP_SEQ_TAKE(n, seq)
n

The number of elements to be extracted. Valid values range from 0 to CHAOS_PP_SEQ_SIZE(seq).

[seq]

The sequence from which elements are to be extracted.

Usage - C/C++ Specific 

CHAOS_PP_SEQ_TAKE(n, seq)
n

The number of elements to be extracted. Valid values range from 0 to CHAOS_PP_SEQ_SIZE(seq).

seq

The sequence from which elements are to be extracted.

Remarks

This macro expands to a sequence containing the first n elements of seq.

Sample Code

#include <chaos/preprocessor/config.h>
#include <chaos/preprocessor/seq/take.h>

#define A (a)(b)(c)(d)(e)

CHAOS_PP_SEQ_TAKE(3, A) // (a)(b)(c)
CHAOS_PP_SEQ_TAKE(0, A) //

#if CHAOS_PP_VARIADICS

CHAOS_PP_SEQ_TAKE(2, (a, b)(c)(d, e, f)) // (a, b)(c)
CHAOS_PP_SEQ_TAKE(0,) //

#endif

CHAOS_PP_SEQ_TAKE_ID

The CHAOS_PP_SEQ_TAKE_ID macro indirectly expands to CHAOS_PP_SEQ_TAKE.

Usage 

CHAOS_PP_SEQ_TAKE_ID()

Remarks

This macro effectively allows CHAOS_PP_SEQ_TAKE to pass through itself without becoming disabled.

CHAOS_PP_SEQ_TAKE_

The CHAOS_PP_SEQ_TAKE_ macro expands to a λ expression representing CHAOS_PP_SEQ_TAKE.

Usage - C99 Specific 

CHAOS_PP_SEQ_TAKE_

Remarks

This macro is a shorthand form of the following λ binding: 

CHAOS_PP_LAMBDA(CHAOS_PP_SEQ_TAKE_ID)()
