The logical/nor.h header defines a logical NOR operator.

Usage

#include <chaos/preprocessor/logical/nor.h>

Contents

  • CHAOS_PP_NOR

CHAOS_PP_NOR

The CHAOS_PP_NOR macro expands to the logical NOR of its operands.

Usage 

CHAOS_PP_NOR(p)(q)
p

The left operand. Valid values range from 0 to CHAOS_PP_LIMIT_MAG.

q

The right operand. Valid values range from 0 to CHAOS_PP_LIMIT_MAG.

Remarks

If p and q are both 0, this macro expands to 1. Otherwise, it expands to 0.

This macro performs a Boolean conversion on each operand. If this conversion is undesirable, CHAOS_PP_BITNOR may be used instead.

This macro is lazy. If p is non-zero, q is not evaluated.

See Also

  • CHAOS_PP_BITNOR

Sample Code

#include <chaos/preprocessor/debug/failure.h>
#include <chaos/preprocessor/logical/nor.h>

CHAOS_PP_NOR(5)(7) // 0
CHAOS_PP_NOR(0)(0) // 1

CHAOS_PP_NOR(10)(CHAOS_PP_FAILURE()) // 0

CHAOS_PP_NOR_ID

The CHAOS_PP_NOR_ID macro indirectly expands to CHAOS_PP_NOR.

Usage 

CHAOS_PP_NOR_ID()

Remarks

This macro exists only for uniformity with other macros that require indirect invocation.

CHAOS_PP_NOR_

The CHAOS_PP_NOR_ macro expands to a λ expression representing CHAOS_PP_NOR.

Usage - C99 Specific 

CHAOS_PP_NOR_

Remarks

This macro is a shorthand form of the following λ binding: 

CHAOS_PP_LAMBDA(CHAOS_PP_NOR)
