The logical/compl.h header defines a bit complement operator.

Usage

#include <chaos/preprocessor/logical/compl.h>

Contents

  • CHAOS_PP_COMPL

CHAOS_PP_COMPL

The CHAOS_PP_COMPL macro expands to the bit complement of its operand.

Usage 

CHAOS_PP_COMPL(bit)
bit

The value whose bit complement is to be produced. Valid values are 0 and 1.

Remarks

If bit is 0, this macro expands to 1. Otherwise, it expands to 0.

This macro does not perform a Boolean conversion on its operand. If this conversion is desirable, CHAOS_PP_NOT may be used instead.

See Also

  • CHAOS_PP_NOT

Sample Code

#include <chaos/preprocessor/logical/compl.h>

CHAOS_PP_COMPL(1) // 0
CHAOS_PP_COMPL(0) // 1

CHAOS_PP_COMPL(CHAOS_PP_COMPL(1)) // 1

CHAOS_PP_COMPL_ID

The CHAOS_PP_COMPL_ID macro indirectly expands to CHAOS_PP_COMPL.

Usage 

CHAOS_PP_COMPL_ID()

Remarks

This macro exists only for uniformity with other macros that require indirect invocation.

CHAOS_PP_COMPL_

The CHAOS_PP_COMPL_ macro expands to a λ expression representing CHAOS_PP_COMPL.

Usage - C99 Specific 

CHAOS_PP_COMPL_

Remarks

This macro is a shorthand form of the following λ binding: 

CHAOS_PP_LAMBDA(CHAOS_PP_COMPL)
