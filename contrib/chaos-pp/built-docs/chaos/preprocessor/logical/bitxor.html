CHAOS_PP_BITXOR

The CHAOS_PP_BITXOR macro expands to the logical XOR of its operands.

Usage 

CHAOS_PP_BITXOR(x)(y)
x

The left operand. Valid values are 0 and 1.

y

The right operand. Valid values are 0 and 1.

Remarks

If either x or y is 1 exclusively, this macro expands to 1. Otherwise, it expands to 0.

This macro does not perform Boolean conversions on its operands. If this conversion is desirable, CHAOS_PP_XOR may be used instead.

See Also

  • CHAOS_PP_XOR

Sample Code

#include <chaos/preprocessor/debug/failure.h>
#include <chaos/preprocessor/logical/bitxor.h>

CHAOS_PP_BITXOR(0)(1) // 1
CHAOS_PP_BITXOR(1)(1) // 0

CHAOS_PP_BITXOR_ID

The CHAOS_PP_BITXOR_ID macro indirectly expands to CHAOS_PP_BITXOR.

Usage 

CHAOS_PP_BITXOR_ID()

Remarks

This macro exists only for uniformity with other macros that require indirect invocation.

CHAOS_PP_BITXOR_

The CHAOS_PP_BITXOR_ macro expands to a λ expression representing CHAOS_PP_BITXOR.

Usage - C99 Specific 

CHAOS_PP_BITXOR_

Remarks

This macro is a shorthand form of the following λ binding: 

CHAOS_PP_LAMBDA(CHAOS_PP_BITXOR)
