The logical/nand.h header defines a logical NAND operator.

Usage

#include <chaos/preprocessor/logical/nand.h>

Contents

  • CHAOS_PP_NAND

CHAOS_PP_NAND

The CHAOS_PP_NAND macro expands to the logical NAND of its operands.

Usage 

CHAOS_PP_NAND(p)(q)
p

The left operand. Valid values range from 0 to CHAOS_PP_LIMIT_MAG.

q

The right operand. Valid values range from 0 to CHAOS_PP_LIMIT_MAG.

Remarks

If both p and q are non-zero, this macro expands to 0. Otherwise, it expands to 1.

This macro performs a Boolean conversion on each operand. If this conversion is undesirable, CHAOS_PP_BITNAND may be used instead.

This macro is lazy. If p is 0, q is not evaluated.

See Also

  • CHAOS_PP_BITNAND

Sample Code

#include <chaos/preprocessor/debug/failure.h>
#include <chaos/preprocessor/logical/nand.h>

CHAOS_PP_NAND(5)(7) // 0
CHAOS_PP_NAND(3)(0) // 1

CHAOS_PP_NAND(0)(CHAOS_PP_FAILURE()) // 1

CHAOS_PP_NAND_ID

The CHAOS_PP_NAND_ID macro indirectly expands to CHAOS_PP_NAND.

Usage 

CHAOS_PP_NAND_ID()

Remarks

This macro exists only for uniformity with other macros that require indirect invocation.

CHAOS_PP_NAND_

The CHAOS_PP_NAND_ macro expands to a λ expression representing CHAOS_PP_NAND.

Usage - C99 Specific 

CHAOS_PP_NAND_

Remarks

This macro is a shorthand form of the following λ binding: 

CHAOS_PP_LAMBDA(CHAOS_PP_NAND)
