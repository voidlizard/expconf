The debug/failure.h header defines a macro that produces an error if it is invoked.

Usage

#include <chaos/preprocessor/debug/failure.h>

Contents

  • CHAOS_PP_FAILURE

CHAOS_PP_FAILURE

The CHAOS_PP_FAILURE macro intentionally produces an error if invoked.

Usage 

CHAOS_PP_FAILURE()

See Also

  • CHAOS_PP_ASSERT
  • CHAOS_PP_ASSERT_MSG

CHAOS_PP_FAILURE_ID

The CHAOS_PP_FAILURE_ID macro indirectly expands to CHAOS_PP_FAILURE.

Usage 

CHAOS_PP_FAILURE_ID()

Remarks

This macro exists only for uniformity with other macros that require indirect invocation.

CHAOS_PP_FAILURE_

The CHAOS_PP_FAILURE_ macro expands to a λ expression representing CHAOS_PP_FAILURE.

Usage - C99 Specific 

CHAOS_PP_FAILURE_

Remarks

This macro is a shorthand form of the following λ binding: 

CHAOS_PP_LAMBDA(CHAOS_PP_FAILURE)
