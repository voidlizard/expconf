CHAOS_PP_SUB_MACHINE

The CHAOS_PP_SUB_MACHINE macro computes the difference between two values.

Usage - C99 Specific 

CHAOS_PP_SUB_MACHINE(x, y)
x

The minuend. Valid values range from 0 to CHAOS_PP_LIMIT_MAG.

y

The subtrahend. Valid values range from 0 to CHAOS_PP_LIMIT_MAG.

Remarks

This macro expands to the difference between x and y. If the result is less than 0, it is saturated to 0.

This macro behaves identically to CHAOS_PP_SUB except that it is implemented with the continuation machine, which can yield significant performance improvements.

This macro may not be used inside a continuation machine instruction. However, the instruction upon which this macro is built, (0xchaos)(0xsub), may be used inside a continuation. 

CHAOS_PP_MACHINE(, 34, (0xchaos)(0xsub), 12, (0xchaos)(0xstop),) // 22

See Also

Sample Code

#include <chaos/preprocessor/arithmetic/machine/sub.h>
#include <chaos/preprocessor/config.h>

#if CHAOS_PP_VARIADICS

CHAOS_PP_SUB_MACHINE(55, 23) // 32

#endif

CHAOS_PP_SUB_MACHINE_ID

The CHAOS_PP_SUB_MACHINE_ID macro indirectly expands to CHAOS_PP_SUB_MACHINE.

Usage - C99 Specific 

CHAOS_PP_SUB_MACHINE_ID()

Remarks

This macro exists only for uniformity with other macros that require indirect invocation.

CHAOS_PP_SUB_MACHINE_

The CHAOS_PP_SUB_MACHINE_ macro expands to a λ expression representing CHAOS_PP_SUB_MACHINE.

Usage - C99 Specific 

CHAOS_PP_SUB_MACHINE_

Remarks

This macro is a shorthand form of the following λ binding: 

CHAOS_PP_LAMBDA(CHAOS_PP_SUB_MACHINE)
