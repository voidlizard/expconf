The arithmetic/dec.h header defines a saturated decrementation operator.

Usage

#include <chaos/preprocessor/arithmetic/dec.h>

Contents

  • CHAOS_PP_DEC

CHAOS_PP_DEC

The CHAOS_PP_DEC macro decrements a number.

Usage 

CHAOS_PP_DEC(x)
x

The value to be decremented. Valid values range from 0 to CHAOS_PP_LIMIT_MAG.

Remarks

If x is 0, this macro expands to 0. Otherwise, it expands to one less than x.

See Also

  • CHAOS_PP_ARBITRARY_DEC
  • CHAOS_PP_ARBITRARY_INC
  • CHAOS_PP_INC

Sample Code

#include <chaos/preprocessor/arithmetic/dec.h>

CHAOS_PP_DEC(0) // 0
CHAOS_PP_DEC(5) // 4

CHAOS_PP_DEC_ID

The CHAOS_PP_DEC_ID macro indirectly expands to CHAOS_PP_DEC.

Usage 

CHAOS_PP_DEC_ID()

Remarks

This macro exists only for uniformity with other macros that require indirect invocation.

CHAOS_PP_DEC_

The CHAOS_PP_DEC_ macro expands to a λ expression representing CHAOS_PP_DEC.

Usage - C99 Specific 

CHAOS_PP_DEC_

Remarks

This macro is a shorthand form of the following λ binding: 

CHAOS_PP_LAMBDA(CHAOS_PP_DEC)
