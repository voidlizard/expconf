Usage

#include <chaos/preprocessor/array/elem.h>

CHAOS_PP_ARRAY_ELEM

The CHAOS_PP_ARRAY_ELEM macro extracts an element from an array.

Usage 

CHAOS_PP_ARRAY_ELEM(i, array)
i

The zero-based index of the element to be extracted. Valid values range from 0 to CHAOS_PP_LIMIT_TUPLE - 1.

array

The array from which an element is to be extracted.

Remarks

This macro expands to the i-th element in array. If i is greater than or equal to the size of array, this macro expands to nothing.

Sample Code

#include <chaos/preprocessor/array/elem.h>

#define DATA() (3, (a, b, c))

CHAOS_PP_ARRAY_ELEM(0, DATA()) // a
CHAOS_PP_ARRAY_ELEM(1, DATA()) // b
CHAOS_PP_ARRAY_ELEM(9, DATA()) //

CHAOS_PP_ARRAY_ELEM_ID

The CHAOS_PP_ARRAY_ELEM_ID macro indirectly expands to CHAOS_PP_ARRAY_ELEM.

Usage 

CHAOS_PP_ARRAY_ELEM_ID()

Remarks

This macro effectively allows CHAOS_PP_ARRAY_ELEM to pass through itself without becoming disabled.

CHAOS_PP_ARRAY_ELEM_

The CHAOS_PP_ARRAY_ELEM_ macro expands to a λ expression representing CHAOS_PP_ARRAY_ELEM.

Usage - C99 Specific 

CHAOS_PP_ARRAY_ELEM_

Remarks

This macro is a shorthand form of the following λ binding: 

CHAOS_PP_LAMBDA(CHAOS_PP_ARRAY_ELEM_ID)()
