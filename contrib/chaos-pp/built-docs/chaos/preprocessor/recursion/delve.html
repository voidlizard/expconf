The recursion/delve.h header defines a mechanism used to count the number of scans that an argument undergoes as it travels through a macro.

Usage

#include <chaos/preprocessor/recursion/delve.h>

Contents

  • CHAOS_PP_DELVE
  • CHAOS_PP_HALT

CHAOS_PP_DELVE

The CHAOS_PP_DELVE macro counts the number of scans that it undergoes.

Usage 

CHAOS_PP_DELVE()

Remarks

This macro counts the number of scans that it undergoes while passing through other macros. It finally expands to that count when the context created by CHAOS_PP_HALT is detected.

See Also

  • CHAOS_PP_HALT

Sample Code

#include <chaos/preprocessor/recursion/delve.h>

#define A(x) B(x)
#define B(x) C(x)
#define C(x) x

CHAOS_PP_HALT(
    A(CHAOS_PP_DELVE())
    B(CHAOS_PP_DELVE())
    C(CHAOS_PP_DELVE())
)
// 4 3 2

CHAOS_PP_DELVE_ID

The CHAOS_PP_DELVE_ID macro indirectly expands to CHAOS_PP_DELVE.

Usage 

CHAOS_PP_DELVE_ID()

Remarks

This macro exists only for uniformity with other macros that require indirect invocation.

CHAOS_PP_DELVE_

The CHAOS_PP_DELVE_ macro expands to a λ expression representing CHAOS_PP_DELVE.

Usage - C99 Specific 

CHAOS_PP_DELVE_

Remarks

This macro is a shorthand form of the following λ binding: 

CHAOS_PP_LAMBDA(CHAOS_PP_DELVE)

CHAOS_PP_HALT

The CHAOS_PP_HALT macro is an identity macro used to create a context that causes invocations of CHAOS_PP_DELVE to stop counting.

Usage - C99 Specific 

CHAOS_PP_HALT(...)
[...]

The argument possibly containing invocations of CHAOS_PP_DELVE to be halted.

Usage - C/C++ Specific 

CHAOS_PP_HALT(x)
x

The argument possibly containing invocations of CHAOS_PP_DELVE to be halted.

Remarks

This macro expands to x or __VA_ARGS__ causing any invocations of CHAOS_PP_DELVE therein to expand to their current counts.

See Also

  • CHAOS_PP_DELVE

Sample Code

#include <chaos/preprocessor/recursion/delve.h>

#define A(x) B(x, x)
#define B(x, y) x C(x)
#define C(x) x

CHAOS_PP_HALT(A(CHAOS_PP_DELVE())) // 3 4

CHAOS_PP_HALT_ID

The CHAOS_PP_HALT_ID macro indirectly expands to CHAOS_PP_HALT.

Usage 

CHAOS_PP_HALT_ID()

Remarks

This macro effectively allows CHAOS_PP_HALT to pass through itself without becoming disabled.

CHAOS_PP_HALT_

The CHAOS_PP_HALT_ macro expands to a λ expression representing CHAOS_PP_HALT.

Usage - C99 Specific 

CHAOS_PP_HALT_

Remarks

This macro is a shorthand form of the following λ binding: 

CHAOS_PP_LAMBDA(CHAOS_PP_HALT_ID)()
