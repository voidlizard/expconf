#include <chaos/preprocessor/recursion/block.h>

CHAOS_PP_BLOCK

The CHAOS_PP_BLOCK macro facilitates the deferral of a macro invocation through a variable number of scans.

Usage 

CHAOS_PP_BLOCK(n, macro)
n

The number of scans through which the invocation of macro is to be deferred. Valid values range from 0 to CHAOS_PP_LIMIT_BLOCK.

macro

The macro name of a macro invocation to be deferred.

Remarks

This macro prevents macro from expanding during the first n scans against arguments that exist following the invocation of CHAOS_PP_BLOCK. For example, 

CHAOS_PP_BLOCK(2, MACRO)(...)

causes the invocation of MACRO to be delayed through two scans.

CHAOS_PP_BLOCK_ID

The CHAOS_PP_BLOCK_ID macro indirectly expands to CHAOS_PP_BLOCK.

Usage 

CHAOS_PP_BLOCK_ID()

Remarks

This macro exists only for uniformity with other macros that require indirect invocation.

CHAOS_PP_BLOCK_

The CHAOS_PP_BLOCK_ macro expands to a λ expression representing CHAOS_PP_BLOCK.

Usage - C99 Specific 

CHAOS_PP_BLOCK_

Remarks

This macro is a shorthand form of the following λ binding: 

CHAOS_PP_LAMBDA(CHAOS_PP_BLOCK)
