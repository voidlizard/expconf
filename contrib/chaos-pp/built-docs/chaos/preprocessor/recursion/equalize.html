The recursion/equalize.h header defines a macro used to equalize the number of scans on an argument.

Usage

#include <chaos/preprocessor/recursion/equalize.h>

Contents

  • CHAOS_PP_EQUALIZE

CHAOS_PP_EQUALIZE

The CHAOS_PP_EQUALIZE macro is used to equalize the number of scans on an argument.

Usage - C99 Specific 

CHAOS_PP_EQUALIZE(n)(x)
n

The number of scans to which x should be subjected. Valid values range from 1 to CHAOS_PP_LIMIT_EQUALIZE.

[x]

The argument to which scans are to be applied.

Usage - C/C++ Specific 

CHAOS_PP_EQUALIZE(n)(x)
n

The number of scans to which x should be subjected. Valid values range from 1 to CHAOS_PP_LIMIT_EQUALIZE.

x

The argument to which scans are to be applied.

Remarks

This macro expands to x after applying n scans to it.

This macro should be used only for the purposes of equalization. Specifically, x may not be an active argument.

Sample Code

#include <chaos/preprocessor/recursion/delve.h>
#include <chaos/preprocessor/recursion/equalize.h>

#define A(p) B(p)
#define B(p) p C(p)
#define C(p) p

CHAOS_PP_HALT(A(CHAOS_PP_DELVE())) // 3 4

#define X(p) Y(p)
#define Y(p) CHAOS_PP_EQUALIZE(1)(p) Z(p)
#define Z(p) p

CHAOS_PP_HALT(X(CHAOS_PP_DELVE())) // 4 4

CHAOS_PP_EQUALIZE_ID

The CHAOS_PP_EQUALIZE_ID macro indirectly expands to CHAOS_PP_EQUALIZE.

Usage 

CHAOS_PP_EQUALIZE_ID()

Remarks

This macro exists only for uniformity with other macros that require indirect invocation.

CHAOS_PP_EQUALIZE_

The CHAOS_PP_EQUALIZE_ macro expands to a λ expression representing CHAOS_PP_EQUALIZE.

Usage - C99 Specific 

CHAOS_PP_EQUALIZE_

Remarks

This macro is a shorthand form of the following λ binding: 

CHAOS_PP_LAMBDA(CHAOS_PP_EQUALIZE)
