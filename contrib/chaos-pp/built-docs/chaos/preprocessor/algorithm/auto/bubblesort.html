The algorithm/auto/bubblesort.h header defines an algorithm that sorts a generic data structure.

Usage

#include <chaos/preprocessor/algorithm/auto/bubblesort.h>

Contents

  • CHAOS_PP_AUTO_BUBBLESORT

CHAOS_PP_AUTO_BUBBLESORT

The CHAOS_PP_AUTO_BUBBLESORT macro sorts its second argument according to the predicate specified by its first argument.

Usage - C99 Specific 

CHAOS_PP_AUTO_BUBBLESORT(pred, g)
pred

The predicate that determines the ordering between one element and another. This predicate must be either a λ expression or a macro name that refers to a macro with a signature compatible with the following pseudo-signature: 

pred(s, x†, y†)

† The pseudo-parameters x and y must each be the number of parameters that corresponds to the number of sub-elements in each element of g.

g

The generic data structure to be sorted.

Usage - C99 Specific 

CHAOS_PP_AUTO_BUBBLESORT(pred, g, ...)
pred

The predicate that determines the ordering between one element and another. This predicate must be either a λ expression or a macro name that refers to a macro with a signature compatible with the following pseudo-signature: 

pred(s, x†, y†, data‡)

† The pseudo-parameters x and y must each be the number of parameters that corresponds to the number of sub-elements in each element of g.

‡ The data pseudo-parameter must be either ... or the number of parameters that corresponds to the number of arguments in the auxiliary data, __VA_ARGS__.

g

The generic data structure to be sorted.

[...]

Auxiliary data to be passed through the algorithm to the predicate.

Remarks

The CHAOS_PP_AUTO_BUBBLESORT macro is a higher-order thunk that deduces the current higher-order entry point and invokes CHAOS_PP_BUBBLESORT.

This macro may not be used if bypass semantics are in effect.

CHAOS_PP_AUTO_BUBBLESORT_ID

The CHAOS_PP_AUTO_BUBBLESORT_ID macro indirectly expands to CHAOS_PP_AUTO_BUBBLESORT.

Usage - C99 Specific 

CHAOS_PP_AUTO_BUBBLESORT_ID()

Remarks

This macro effectively allows CHAOS_PP_AUTO_BUBBLESORT to be passed around without expansion.

CHAOS_PP_AUTO_BUBBLESORT_

The CHAOS_PP_AUTO_BUBBLESORT_ macro expands to a λ expression representing CHAOS_PP_AUTO_BUBBLESORT.

Usage - C99 Specific 

CHAOS_PP_AUTO_BUBBLESORT_

Remarks

This macro is a shorthand form of the following λ binding: 

CHAOS_PP_LAMBDA(CHAOS_PP_AUTO_BUBBLESORT_ID)()
