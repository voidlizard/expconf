The algorithm/auto/for_each_product.h header defines an algorithm that repeats a user-specified target for each Cartesian product of elements in a sequence of generic data structures.

Usage

#include <chaos/preprocessor/algorithm/auto/for_each_product.h>

Contents

  • CHAOS_PP_AUTO_FOR_EACH_PRODUCT

CHAOS_PP_AUTO_FOR_EACH_PRODUCT

The CHAOS_PP_AUTO_FOR_EACH_PRODUCT macro repeats the target specified by its first argument for each Cartesian product of elements in its second argument.

Usage - C99 Specific 

CHAOS_PP_AUTO_FOR_EACH_PRODUCT(macro, seq)
macro

The target that is to be repeated for each Cartesian product in seq. This target must be either a λ expression or a macro name that refers to a macro with a signature compatible with the following pseudo-signature: 

macro(s, product)
[seq]

The sequence of generic data structures from which Cartesian products are to be generated.

Usage - C99 Specific 

CHAOS_PP_AUTO_FOR_EACH_PRODUCT(macro, seq, ...)
macro

The target that is to be repeated for each Cartesian product in seq. This target must be either a λ expression or a macro name that refers to a macro with a signature compatible with the following pseudo-signature: 

macro(s, product, data†)

† The data pseudo-parameter must be either ... or the number of parameters that corresponds to the number of arguments in the auxiliary data, __VA_ARGS__.

[seq]

The sequence of generic data structures from which Cartesian products are to be generated.

[...]

Auxiliary data to be passed through the algorithm to the target.

Remarks

The CHAOS_PP_AUTO_FOR_EACH_PRODUCT macro is a higher-order thunk that deduces the current higher-order entry point and invokes CHAOS_PP_FOR_EACH_PRODUCT.

This macro may not be used if bypass semantics are in effect.

CHAOS_PP_AUTO_FOR_EACH_PRODUCT_ID

The CHAOS_PP_AUTO_FOR_EACH_PRODUCT_ID macro indirectly expands to CHAOS_PP_AUTO_FOR_EACH_PRODUCT.

Usage - C99 Specific 

CHAOS_PP_AUTO_FOR_EACH_PRODUCT_ID()

Remarks

This macro effectively allows CHAOS_PP_AUTO_FOR_EACH_PRODUCT to be passed around without expansion.

CHAOS_PP_AUTO_FOR_EACH_PRODUCT_

The CHAOS_PP_AUTO_FOR_EACH_PRODUCT_ macro expands to a λ expression representing CHAOS_PP_AUTO_FOR_EACH_PRODUCT.

Usage - C99 Specific 

CHAOS_PP_AUTO_FOR_EACH_PRODUCT_

Remarks

This macro is a shorthand form of the following λ binding: 

CHAOS_PP_LAMBDA(CHAOS_PP_AUTO_FOR_EACH_PRODUCT_ID)()
