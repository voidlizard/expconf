The algorithm/elem.h header defines an algorithm that extracts an element from a generic data structure

Usage

#include <chaos/preprocessor/algorithm/elem.h>

Contents

  • CHAOS_PP_ELEM

CHAOS_PP_ELEM

The CHAOS_PP_ELEM macro extracts an element from a generic data structure.

Usage 

CHAOS_PP_ELEM(i, g)
i

The zero-based index of the element to be extracted. Valid values range from 0 to CHAOS_PP_SIZE(g) - 1.

g

The non-nil generic data structure from which an element is to be extracted.

Remarks

This macro extracts the i-th element of g.

The efficiency of the algorithm's use of the recursion backend can be described in terms of the number of recursion steps required to complete the algorithm. The asymptotic upper and lower bounds are Θ(n+1), where n is i.

This macro uses bypass semantics. If bypass semantics are already in effect, CHAOS_PP_ELEM_BYPASS must be used instead.

Sample Code

#include <chaos/preprocessor/algorithm/elem.h>
#include <chaos/preprocessor/seq/core.h>

#define DATA (CHAOS_PP_SEQ) (a)(b)(c)

CHAOS_PP_ELEM(0, DATA) // a
CHAOS_PP_ELEM(2, DATA) // c

CHAOS_PP_ELEM_ID

The CHAOS_PP_ELEM_ID macro indirectly expands to CHAOS_PP_ELEM.

Usage 

CHAOS_PP_ELEM_ID()

Remarks

This macro effectively allows CHAOS_PP_ELEM to pass through itself without becoming disabled.

CHAOS_PP_ELEM_

The CHAOS_PP_ELEM_ macro expands to a λ expression representing CHAOS_PP_ELEM.

Usage - C99 Specific 

CHAOS_PP_ELEM_

Remarks

This macro is a shorthand form of the following λ binding: 

CHAOS_PP_LAMBDA(CHAOS_PP_ELEM_ID)()

CHAOS_PP_ELEM_BYPASS

The CHAOS_PP_ELEM_BYPASS macro behaves identically to CHAOS_PP_ELEM except that it is parametized by the bypass recursion state.

Usage 

CHAOS_PP_ELEM_BYPASS(s, i, g)
s

The current bypass recursion state.

i

The zero-based index of the element to be extracted. Valid values range from 0 to CHAOS_PP_SIZE(g) - 1.

g

The non-nil generic data structure from which an element is to be extracted.

Remarks

If bypass semantics are already in effect, this macro must be used instead of CHAOS_PP_ELEM.

CHAOS_PP_ELEM_BYPASS_ID
