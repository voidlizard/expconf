The algorithm/reverse.h header defines an algorithm that reverses the elements in a generic data structure.

Usage

#include <chaos/preprocessor/algorithm/reverse.h>

Contents

  • CHAOS_PP_REVERSE

CHAOS_PP_REVERSE

The CHAOS_PP_REVERSE macro reverses the elements in its argument.

Usage 

CHAOS_PP_REVERSE(g)
g

The generic data structure whose elements are to be reversed.

Remarks

This macro expands to a generic data structure of the same type as g containing the elements of g in reverse order.

The efficiency of the algorithm's use of the recursion backend can be described in terms of the number of recursion steps required to complete the algorithm. The asymptotic upper and lower bounds are Θ(n+1), where n is the number of elements in g.

This macro uses bypass semantics. If bypass semantics are already in effect, CHAOS_PP_REVERSE_BYPASS must be used instead.

Sample Code

#include <chaos/preprocessor/algorithm/reverse.h>
#include <chaos/preprocessor/seq/core.h>

CHAOS_PP_REVERSE((CHAOS_PP_SEQ) (a)(b)(c))
// (CHAOS_PP_SEQ) (c)(b)(a)

CHAOS_PP_REVERSE_ID

The CHAOS_PP_REVERSE_ID macro indirectly expands to CHAOS_PP_REVERSE.

Usage 

CHAOS_PP_REVERSE_ID()

Remarks

This macro effectively allows CHAOS_PP_REVERSE to pass through itself without becoming disabled.

CHAOS_PP_REVERSE_

The CHAOS_PP_REVERSE_ macro expands to a λ expression representing CHAOS_PP_REVERSE.

Usage - C99 Specific 

CHAOS_PP_REVERSE_

Remarks

This macro is a shorthand form of the following λ binding: 

CHAOS_PP_LAMBDA(CHAOS_PP_REVERSE_ID)()

CHAOS_PP_REVERSE_BYPASS

The CHAOS_PP_REVERSE_BYPASS macro behaves identically to CHAOS_PP_REVERSE except that it is parametized by the bypass recursion state.

Usage 

CHAOS_PP_REVERSE_BYPASS(s, g)
s

The current bypass recursion state.

g

The generic data structure whose elements are to be reversed.

Remarks

If bypass semantics are already in effect, this macro must be used instead of CHAOS_PP_REVERSE.

CHAOS_PP_REVERSE_BYPASS_ID

The CHAOS_PP_REVERSE_BYPASS_ID macro indirectly expands to CHAOS_PP_REVERSE_BYPASS.

Usage 

CHAOS_PP_REVERSE_BYPASS_ID()

Remarks

This macro effectively allows CHAOS_PP_REVERSE_BYPASS to pass through itself without becoming disabled.

CHAOS_PP_REVERSE_BYPASS_

Usage - C99 Specific 

CHAOS_PP_REVERSE_BYPASS_

Remarks
