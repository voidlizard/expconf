The tuple/as_args.h header defines a macro used to convert a tuple to a representation suitable for use as an argument list.

Usage

#include <chaos/preprocessor/tuple/as_args.h>

Contents

  • CHAOS_PP_TUPLE_AS_ARGS

CHAOS_PP_TUPLE_AS_ARGS

The CHAOS_PP_TUPLE_AS_ARGS macro converts its argument to a representation suitable for use as an argument list.

Usage - C99 Specific 

CHAOS_PP_TUPLE_AS_ARGS(tuple)
[tuple]

The tuple to be converted.

Remarks

If tuple is nil, this macro expands to (). Otherwise, it expands to tuple.

This macro operates on the nil tuple representation used by the core tuple operations defined in <chaos/preprocessor/tuple/core.h>. Specifically, () is a one-element tuple, not a zero-element tuple.

Sample Code

#include <chaos/preprocessor/config.h>
#include <chaos/preprocessor/recursion/expr.h>
#include <chaos/preprocessor/tuple/as_args.h>
#include <chaos/preprocessor/tuple/size.h>

#if CHAOS_PP_VARIADICS

#define APPLY(prefix, tuple) \
    CHAOS_PP_CAT(prefix, CHAOS_PP_TUPLE_QUICK_SIZE(tuple)) \
    CHAOS_PP_TUPLE_AS_ARGS(tuple) \
    /**/

#define MACRO_0()
#define MACRO_1(a) a
#define MACRO_2(a, b) a * b
#define MACRO_3(a, b, c) a * b * c

#define MACRO(args) CHAOS_PP_EXPR(APPLY(MACRO_, args))

MACRO()       //
MACRO((1))    // 1
MACRO((1, 2)) // 1 * 2

#endif

CHAOS_PP_TUPLE_AS_ARGS_ID

The CHAOS_PP_TUPLE_AS_ARGS_ID macro indirectly expands to CHAOS_PP_TUPLE_AS_ARGS.

Usage - C99 Specific 

CHAOS_PP_TUPLE_AS_ARGS_ID()

Remarks

This macro effectively allows CHAOS_PP_TUPLE_AS_ARGS to pass through itself without becoming disabled.

CHAOS_PP_TUPLE_AS_ARGS_

The CHAOS_PP_TUPLE_AS_ARGS_ macro expands to a λ expression representing CHAOS_PP_TUPLE_AS_ARGS.

Usage - C99 Specific 

CHAOS_PP_TUPLE_AS_ARGS_

Remarks

This macro is a shorthand form of the following λ binding: 

CHAOS_PP_LAMBDA(CHAOS_PP_TUPLE_AS_ARGS_ID)()
