The detection/is_numeric.h header defines a macro that detects whether its argument is a number.

Usage

#include <chaos/preprocessor/detection/is_numeric.h>

Contents

  • CHAOS_PP_IS_NUMERIC

CHAOS_PP_IS_NUMERIC

The CHAOS_PP_IS_NUMERIC macro detects whether its argument is a number.

Usage - C99 Specific 

CHAOS_PP_IS_NUMERIC(x)
[x]

The argument to be tested.

Usage - C/C++ Specific 

CHAOS_PP_IS_NUMERIC(x)
x

The argument to be tested.

Remarks

If x is a number between 0 and CHAOS_PP_LIMIT_MAG, this macro expands to 1. Otherwise, it expands to 0.

Input to this macro is restricted to numbers and identifiers. Specifically, operators are invalid input. 

CHAOS_PP_IS_NUMERIC(+) // invalid

This restriction is relaxed somewhat in C99. In C99, x may be empty or may be a parenthetic expression.

Sample Code

#include <chaos/preprocessor/detection/is_numeric.h>

CHAOS_PP_IS_NUMERIC(ABC) // 0
CHAOS_PP_IS_NUMERIC(123) // 1

CHAOS_PP_IS_NUMERIC_ID

The CHAOS_PP_IS_NUMERIC_ID macro indirectly expands to CHAOS_PP_IS_NUMERIC.

Usage 

CHAOS_PP_IS_NUMERIC_ID()

Remarks

This macro exists only for uniformity with other macros that require indirect invocation.

CHAOS_PP_IS_NUMERIC_

The CHAOS_PP_IS_NUMERIC_ macro expands to a λ expression representing CHAOS_PP_IS_NUMERIC.

Usage - C99 Specific 

CHAOS_PP_IS_NUMERIC_

Remarks

This macro is a shorthand form of the following λ binding: 

CHAOS_PP_LAMBDA(CHAOS_PP_IS_NUMERIC)
