Usage

#include <chaos/preprocessor/detection/compare.h>

Contents

CHAOS_PP_COMPARE

The CHAOS_PP_COMPARE macro detects whether two registered identifiers are the same.

Usage 

CHAOS_PP_COMPARE(a, b)
a

The first identifier.

b

The second identifier.

Remarks

If a and b are both the same identifier and that identifier is registered, this macro expands to 1. Otherwise, it expands to 0.

The lowercase and uppercase letters, the underscore, and all of the C and C++ keywords are pre-registered by the library. Further identifiers can be registered with a macro definition as follows (where xyz is a placeholder for some identifier): 

#define CHAOS_PP_COMPARE_xyz(x) x

Registration of the identifier ID is prohibited, because that would form CHAOS_PP_COMPARE_ID, which is a library primitive.

See Also

  • CHAOS_PP_PRIMITIVE_COMPARE

Sample Code

#include <chaos/preprocessor/detection/compare.h>

#define A int

CHAOS_PP_COMPARE(A, int)    // 1
CHAOS_PP_COMPARE(A, double) // 0

#define CHAOS_PP_COMPARE_boost(x) x

#define B boost

CHAOS_PP_COMPARE(B, abc)   // 0
CHAOS_PP_COMPARE(B, boost) // 1

CHAOS_PP_COMPARE_ID

The CHAOS_PP_COMPARE_ID macro indirectly expands to CHAOS_PP_COMPARE.

Usage 

CHAOS_PP_COMPARE_ID()

Remarks

This macro exists only for uniformity with other macros that require indirect invocation.

CHAOS_PP_COMPARE_

The CHAOS_PP_COMPARE_ macro expands to a λ expression representing CHAOS_PP_COMPARE.

Usage - C99 Specific 

CHAOS_PP_COMPARE_

Remarks

This macro is a shorthand form of the following λ binding: 

CHAOS_PP_LAMBDA(CHAOS_PP_COMPARE)

CHAOS_PP_PRIMITIVE_COMPARE

The CHAOS_PP_PRIMITIVE_COMPARE macro detects whether two registered identifiers are the same.

Usage 

CHAOS_PP_PRIMITIVE_COMPARE(a, b)
a

The first identifier.

b

The second identifier.

Remarks

This macro behaves identically to CHAOS_PP_COMPARE except that it does not provide the delay necessary to allow its arguments to expand on input.

See Also

  • CHAOS_PP_COMPARE

Sample Code

#include <chaos/preprocessor/detection/compare.h>

#define Object Result

#define CHAOS_PP_COMPARE_Object(x) x

CHAOS_PP_COMPARE(Object, Object)           // 0
CHAOS_PP_PRIMITIVE_COMPARE(Object, Object) // 1
