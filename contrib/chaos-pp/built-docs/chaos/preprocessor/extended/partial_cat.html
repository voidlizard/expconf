The extended/partial_cat.h header defines a concatenation macro that selectively allows each argument to expand on input.

Usage

#include <chaos/preprocessor/extended/partial_cat.h>

CHAOS_PP_PARTIAL

The CHAOS_PP_PARTIAL macro is used as a partial operand to CHAOS_PP_PARTIAL_CAT.

Usage 

CHAOS_PP_PARTIAL

Remarks

By itself, the CHAOS_PP_PARTIAL macro expands to nothing. It exists as a helper macro for CHAOS_PP_PARTIAL_CAT.

See Also

  • CHAOS_PP_PARTIAL_CAT

CHAOS_PP_PARTIAL_CAT

The CHAOS_PP_PARTIAL_CAT macro is a concatenation macro that selectively allows each argument to expand before concatenation.

Usage - C99 Specific 

CHAOS_PP_PARTIAL_CAT(i, j)(CHAOS_PP_PARTIAL a, ... CHAOS_PP_PARTIAL)
i

The argument that determines whether a is allowed to expand. Valid values are 0 and 1.

j

The argument that determines whether __VA_ARGS__ is allowed to expand. Valid values are 0 and 1.

[a]

The left operand of the concatenation.

[...]

The right operand of the concatenation.

Usage - C/C++ Specific 

CHAOS_PP_PARTIAL_CAT(i, j)(CHAOS_PP_PARTIAL a, b CHAOS_PP_PARTIAL)
i

The argument that determines whether a is allowed to expand. Valid values are 0 and 1.

j

The argument that determines whether b is allowed to expand. Valid values are 0 and 1.

a

The left operand of the concatenation.

b

The right operand of the concatenation.

Remarks

This macro concatenates the rightmost preprocessing token of a to the leftmost preprocessing token of b or __VA_ARGS__.

If i is 1, a is allowed to expand prior to concatenation. Otherwise, it is not allowed to expand prior to concatenation. Likewise, if j is 1, b or __VA_ARGS__ is allowed to expand prior to concatenation. Otherwise, it is not allowed to expand prior to concatenation.

Sample Code

#include <chaos/preprocessor/extended/partial_cat.h>

CHAOS_PP_PARTIAL_CAT(0, 1)(
    CHAOS_PP_PARTIAL __LINE__, __LINE__ CHAOS_PP_PARTIAL
)
// __LINE__4

CHAOS_PP_PARTIAL_CAT_ID

The CHAOS_PP_PARTIAL_CAT_ID macro indirectly expands to CHAOS_PP_PARTIAL_CAT.

Usage 

CHAOS_PP_PARTIAL_CAT_ID()

Remarks

This macro exists only for uniformity with other macros that require indirect invocation.

CHAOS_PP_PARTIAL_CAT_II

The CHAOS_PP_PARTIAL_CAT_II macro is a concatenation macro that selectively allows each argument to expand before concatenation.

Usage - C99 Specific 

CHAOS_PP_PARTIAL_CAT_II(i, j)(, a, ...)
i

The argument that determines whether a is allowed to expand. Valid values are 0 and 1.

j

The argument that determines whether __VA_ARGS__ is allowed to expand. Valid values are 0 and 1.

[a]

The left operand of the concatenation.

[...]

The right operand of the concatenation.

Remarks

This macro concatenates the rightmost preprocessing token of a to the leftmost preprocessing token of b or __VA_ARGS__.

If i is 1, a is allowed to expand prior to concatenation. Otherwise, it is not allowed to expand prior to concatenation. Likewise, if j is 1, b or __VA_ARGS__ is allowed to expand prior to concatenation. Otherwise, it is not allowed to expand prior to concatenation.

Note that the first argument of the second argument group must be a placeholder (i.e. empty).
