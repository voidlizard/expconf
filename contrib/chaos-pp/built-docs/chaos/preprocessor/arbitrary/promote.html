The arbitrary/promote.h header defines a macro that converts an integer literal to an arbitrary-precision value.

Usage

#include <chaos/preprocessor/arbitrary/promote.h>

Contents

  • CHAOS_PP_ARBITRARY_PROMOTE

CHAOS_PP_ARBITRARY_PROMOTE

The CHAOS_PP_ARBITRARY_PROMOTE macro converts its argument to an arbitrary-precision value.

Usage 

CHAOS_PP_ARBITRARY_PROMOTE(x)
x

The value to be converted. Valid values range from 0 to CHAOS_PP_LIMIT_MAG. This value may also be prefixed with a parenthesized negation operator.

Remarks

This macro expands to an arbitrary-precision value representing x.

See Also

  • CHAOS_PP_ARBITRARY_DEMOTE

Sample Code

#include <chaos/preprocessor/arbitrary/promote.h>

CHAOS_PP_ARBITRARY_PROMOTE(123)   // (1)(2)(3)
CHAOS_PP_ARBITRARY_PROMOTE((-)10) // ((1)(0))

CHAOS_PP_ARBITRARY_PROMOTE_ID

The CHAOS_PP_ARBITRARY_PROMOTE_ID macro indirectly expands to CHAOS_PP_ARBITRARY_PROMOTE.

Usage 

CHAOS_PP_ARBITRARY_PROMOTE_ID()

Remarks

This macro exists only for uniformity with other macros that require indirect invocation.

CHAOS_PP_ARBITRARY_PROMOTE_

The CHAOS_PP_ARBITRARY_PROMOTE_ macro expands to a λ expression representing CHAOS_PP_ARBITRARY_PROMOTE.

Usage - C99 Specific 

CHAOS_PP_ARBITRARY_PROMOTE_

Remarks

This macro is a shorthand form of the following λ binding: 

CHAOS_PP_LAMBDA(CHAOS_PP_ARBITRARY_PROMOTE)
