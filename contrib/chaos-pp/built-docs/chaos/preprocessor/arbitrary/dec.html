The arbitrary/dec.h header defines an arbitrary-precision decrementation operator.

Usage

#include <chaos/preprocessor/arbitrary/dec.h>

Contents

CHAOS_PP_ARBITRARY_DEC

The CHAOS_PP_ARBITRARY_DEC macro decrements an arbitrary-precision value.

Usage 

CHAOS_PP_ARBITRARY_DEC(x)
x

The arbitrary-precision value to be decremented.

Remarks

This macro results in an arbitrary-precision value one less than x.

See Also

  • CHAOS_PP_ARBITRARY_INC
  • CHAOS_PP_DEC

Sample Code

#include <chaos/preprocessor/arbitrary/dec.h>
#include <chaos/preprocessor/arbitrary/demote.h>
#include <chaos/preprocessor/arbitrary/literal.h>

CHAOS_PP_ARBITRARY_DEMOTE(
    CHAOS_PP_ARBITRARY_DEC(CHAOS_PP_ARBITRARY_LITERAL(1 2 3))
)
// 122

CHAOS_PP_ARBITRARY_DEMOTE(
    CHAOS_PP_ARBITRARY_DEC(CHAOS_PP_ARBITRARY_LITERAL((-)1 2 3))
)
// -124

CHAOS_PP_ARBITRARY_DEMOTE(
    CHAOS_PP_ARBITRARY_DEC(CHAOS_PP_ARBITRARY_LITERAL(0))
)
// -1

CHAOS_PP_ARBITRARY_DEC_ID

The CHAOS_PP_ARBITRARY_DEC_ID macro indirectly expands to CHAOS_PP_ARBITRARY_DEC.

Usage 

CHAOS_PP_ARBITRARY_DEC_ID()

Remarks

This macro exists only for uniformity with other macros that require indirect invocation.

CHAOS_PP_ARBITRARY_DEC_

The CHAOS_PP_ARBITRARY_DEC_ macro expands to a λ expression representing CHAOS_PP_ARBITRARY_DEC.

Usage - C99 Specific 

CHAOS_PP_ARBITRARY_DEC_

Remarks

This macro is a shorthand form of the following λ binding: 

CHAOS_PP_LAMBDA(CHAOS_PP_ARBITRARY_DEC)
