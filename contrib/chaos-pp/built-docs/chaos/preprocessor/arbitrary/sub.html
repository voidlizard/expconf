CHAOS_PP_ARBITRARY_SUB

The CHAOS_PP_ARBITRARY_SUB macro computes the difference between two arbitrary-precision values.

Usage 

CHAOS_PP_ARBITRARY_SUB(x, y)
x

The minuend.

y

The subtrahend.

Remarks

This macro expands to a arbitrary-precision value that represents the difference between x and y.

See Also

  • CHAOS_PP_SUB
  • CHAOS_PP_SUB_MACHINE

Sample Code

#include <chaos/preprocessor/arbitrary/sub.h>
#include <chaos/preprocessor/arbitrary/demote.h>

CHAOS_PP_ARBITRARY_DEMOTE(CHAOS_PP_ARBITRARY_DEMOTE(
    (1)(0)(0)(0)(0),
    (1)(9)(9)
))
// 9801

CHAOS_PP_ARBITRARY_SUB_ID

The CHAOS_PP_ARBITRARY_SUB_ID macro indirectly expands to CHAOS_PP_ARBITRARY_SUB.

Usage 

CHAOS_PP_ARBITRARY_SUB_ID()

Remarks

This macro exists only for uniformity with other macros that require indirect invocation.

CHAOS_PP_ARBITRARY_SUB_

The CHAOS_PP_ARBITRARY_SUB_ macro expands to a λ expression representing CHAOS_PP_ARBITRARY_SUB.

Usage - C99 Specific 

CHAOS_PP_ARBITRARY_SUB_

Remarks

This macro is a shorthand form of the following λ binding: 

CHAOS_PP_LAMBDA(CHAOS_PP_ARBITRARY_SUB)
