The arbitrary/greater.h header defines an arbitrary-precision "greater-than" operator.

Usage

#include <chaos/preprocessor/arbitrary/greater.h>

Contents

  • CHAOS_PP_ARBITRARY_GREATER

CHAOS_PP_ARBITRARY_GREATER

The CHAOS_PP_ARBITRARY_GREATER macro tests whether its first operand is greater than its second operand.

Usage 

CHAOS_PP_ARBITRARY_GREATER(x, y)
x

The first operand.

y

The second operand.

Remarks

If x is greater than y, this macro expands to 1. Otherwise, it expands to 0.

See Also

  • CHAOS_PP_GREATER

Sample Code

#include <chaos/preprocessor/arbitrary/greater.h>

CHAOS_PP_ARBITRARY_GREATER((1)(2)(3), (4)(5)) // 1
CHAOS_PP_ARBITRARY_GREATER((1)(2), (3)(4)(5)) // 0

CHAOS_PP_ARBITRARY_GREATER_ID

The CHAOS_PP_ARBITRARY_GREATER_ID macro indirectly expands to CHAOS_PP_ARBITRARY_GREATER.

Usage 

CHAOS_PP_ARBITRARY_GREATER_ID()

Remarks

This macro exists only for uniformity with other macros that require indirect invocation.

CHAOS_PP_ARBITRARY_GREATER_

The CHAOS_PP_ARBITRARY_GREATER_ macro expands to a λ expression representing CHAOS_PP_ARBITRARY_GREATER.

Usage - C99 Specific 

CHAOS_PP_ARBITRARY_GREATER_

Remarks

This macro is a shorthand form of the following λ binding: 

CHAOS_PP_LAMBDA(CHAOS_PP_ARBITRARY_GREATER)
