The arbitrary/add.h header defines an arbitrary-precision addition operator.

Usage

#include <chaos/preprocessor/arbitrary/add.h>

Contents

  • CHAOS_PP_ARBITRARY_ADD

CHAOS_PP_ARBITRARY_ADD

The CHAOS_PP_ARBITRARY_ADD macro computes the sum of two arbitrary-precision values.

Usage 

CHAOS_PP_ARBITRARY_ADD(x, y)
x

The first addend.

y

The second addend.

Remarks

This macro expands to an arbitrary-precision value that represents the sum of x and y.

See Also

  • CHAOS_PP_ADD
  • CHAOS_PP_ADD_MACHINE

Sample Code

#include <chaos/preprocessor/arbitrary/add.h>
#include <chaos/preprocessor/arbitrary/demote.h>

CHAOS_PP_ARBITRARY_DEMOTE(CHAOS_PP_ARBITRARY_ADD(
    (9)(8)(7)(6)(5)(4)(3)(2)(1)(0)(9)(8)(7)(6)(5)(4)(3)(2)(1)(0),
    (8)(7)(6)(5)(4)(3)(2)(1)(0)(9)(8)(7)(6)(5)(4)(3)(2)(1)(0)(9)
))
// 186419753208641975319

CHAOS_PP_ARBITRARY_ADD_ID

The CHAOS_PP_ARBITRARY_ADD_ID macro indirectly expands to CHAOS_PP_ARBITRARY_ADD.

Usage 

CHAOS_PP_ARBITRARY_ADD_ID()

Remarks

This macro exists only for uniformity with other macros that require indirect invocation.

CHAOS_PP_ARBITRARY_ADD_

The CHAOS_PP_ARBITRARY_ADD_ macro expands to a λ expression representing CHAOS_PP_ARBITRARY_ADD.

Usage - C99 Specific 

CHAOS_PP_ARBITRARY_ADD_

Remarks

This macro is a shorthand form of the following λ binding: 

CHAOS_PP_LAMBDA(CHAOS_PP_ARBITRARY_ADD)
