CHAOS_PP_ARBITRARY_NEG

The CHAOS_PP_ARBITRARY_NEG macro negates an arbitrary-precision value.

Usage 

CHAOS_PP_ARBITRARY_NEG(x)
x

The arbitrary-precision value to be negated.

Remarks

If x is an arbitrary-precision value representing zero, this macro expands to x. Otherwise, it expands to an arbitrary-precision value that represents the negation of x.

Sample Code

#include <chaos/preprocessor/arbitrary/demote.h>
#include <chaos/preprocessor/arbitrary/negate.h>

CHAOS_PP_ARBITRARY_DEMOTE(CHAOS_PP_ARBITRARY_NEG(
    (1)(2)(3)
))
// -123

CHAOS_PP_ARBITRARY_DEMOTE(CHAOS_PP_ARBITRARY_NEG(
    ((1)(2)(3))
))
// 123

CHAOS_PP_ARBITRARY_NEG_ID

The CHAOS_PP_ARBITRARY_NEG_ID macro indirectly expands to CHAOS_PP_ARBITRARY_NEG.

Usage 

CHAOS_PP_ARBITRARY_NEG_ID()

Remarks

This macro exists only for uniformity with other macros that require indirect invocation.

CHAOS_PP_ARBITRARY_NEG_

The CHAOS_PP_ARBITRARY_NEG_ macro expands to a λ expression representing CHAOS_PP_ARBITRARY_NEG.

Usage - C99 Specific 

CHAOS_PP_ARBITRARY_NEG_

Remarks

This macro is a shorthand form of the following λ binding: 

CHAOS_PP_LAMBDA(CHAOS_PP_ARBITRARY_NEG)
