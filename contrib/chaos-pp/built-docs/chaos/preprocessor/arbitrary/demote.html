The arbitrary/demote.h header defines a macro that converts an arbitrary-precision value to an integer literal.

Usage

#include <chaos/preprocessor/arbitrary/demote.h>

Contents

  • CHAOS_PP_ARBITRARY_DEMOTE

CHAOS_PP_ARBITRARY_DEMOTE

The CHAOS_PP_ARBITRARY_DEMOTE macro converts its argument to an integer literal.

Usage 

CHAOS_PP_ARBITRARY_DEMOTE(x)
x

The arbitrary-precision value to be converted.

Remarks

This macro expands to an integer literal represented by x. If x is an arbitrary-precision value representing a negative number, this macro prefixes the result with a unary negation operator (-).

See Also

  • CHAOS_PP_ARBITRARY_PROMOTE

Sample Code

#include <chaos/preprocessor/arbitrary/demote.h>

CHAOS_PP_ARBITRARY_DEMOTE( (1)(2)(3) ) // 123
CHAOS_PP_ARBITRARY_DEMOTE( ((1)(0))  ) // -10

CHAOS_PP_ARBITRARY_DEMOTE_ID

The CHAOS_PP_ARBITRARY_DEMOTE_ID macro indirectly expands to CHAOS_PP_ARBITRARY_DEMOTE.

Usage 

CHAOS_PP_ARBITRARY_DEMOTE_ID()

Remarks

This macro exists only for uniformity with other macros that require indirect invocation.

CHAOS_PP_ARBITRARY_DEMOTE_

The CHAOS_PP_ARBITRARY_DEMOTE_ macro expands to a λ expression representing CHAOS_PP_ARBITRARY_DEMOTE.

Usage - C99 Specific 

CHAOS_PP_ARBITRARY_DEMOTE_

Remarks

This macro is a shorthand form of the following λ binding: 

CHAOS_PP_LAMBDA(CHAOS_PP_ARBITRARY_DEMOTE)
