Welcome to Chaos!

The Chaos Preprocessor Library is a generative metaprogramming framework for C and C++ that builds on the techniques used in the Boost Preprocessor Library. It is primarily intended to push the state-of-the-art in the field of preprocessor metaprogramming. As such, it assumes a strictly conformant preprocessor in an effort to reduce limitations and provide motivation for vendors to fix their broken preprocessors. This reduces the current portability of the library, as many preprocessors have serious conformancy issues. At the same time, it functions as a reference implementation of advanced preprocessor metaprogramming concepts, idioms, and techniques.

If you find Chaos to be interesting, you may find Chaos' sister project, Order, to be interesting as well. Both Order and Chaos are distributed under the Boost Software License.
