A list is a data structure resembling a cons-style singly linked list. A list is either an ellipsis (...), which represents a nil list, or a binary tuple whose first element is an element in the list and whose second element is another list. For example, 

(a, (b, (c, ...)))

is a list containing three elements, a, b, and c.

Lists cannot contain elements with any arity other than unary even in C99. However, they can contain nested lists or other data structures that contain multiple elements. For example, the following is a list of tuples: 

((a, b), ((p, q), ((x, y), ...)))

The list-specific primitives are located in the chaos/preprocessor/list subdirectory.

Using Lists as Generic Data Structures

Lists can be used as generic data structures if the <chaos/preprocessor/list/core.h> header is included. For example, 

#define MACRO(s, x, _) + x

CHAOS_PP_EXPR(CHAOS_PP_FOR_EACH(
    MACRO, (CHAOS_PP_LIST) (x, (y, (z, ...))), ~
))
// + x + y + z
